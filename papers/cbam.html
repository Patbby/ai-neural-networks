CBAM

Convolutional Block Attention Module

A simple yet effective attention module that sequentially infers attention maps along channel and spatial dimensions for adaptive feature refinement

Key Contributions

  • • Sequential channel and spatial attention mechanism
  • • Lightweight module with negligible overhead
  • • Plug-and-play module for any CNN architecture
  • • Demonstrated improvements on ImageNet, COCO, and VOC

CBAM Architecture

Channel Attention Module

Channel attention focuses on "what" is meaningful given an input image. To compute channel attention efficiently, CBAM squeezes the spatial dimension of the input feature map.

Computation Steps:

  1. 1. Apply global average pooling and max pooling
  2. 2. Forward through shared MLP (2 FC layers)
  3. 3. Element-wise sum and sigmoid activation

Mathematical Formulation:

$$M_c(F) = \sigma(\text{MLP}(\text{AvgPool}(F)) + \text{MLP}(\text{MaxPool}(F)))$$

Where:

  • • $\sigma$ = sigmoid activation
  • • MLP has one hidden layer with reduction ratio $r$
  • • Output dimension: $C \times 1 \times 1$

Spatial Attention Module

Spatial attention focuses on "where" is meaningful. This is complementary to channel attention by focusing on spatial locations rather than channels.

Computation Steps:

  1. 1. Apply channel-wise average and max pooling
  2. 2. Concatenate along channel dimension
  3. 3. Apply 7×7 convolution + sigmoid

Mathematical Formulation:

$$M_s(F') = \sigma(f^{7\times7}([\text{AvgPool}(F'); \text{MaxPool}(F')]))$$

Where:

  • • $f^{7\times7}$ = 7×7 convolution operation
  • • Concatenation along channel axis
  • • Output dimension: $1 \times H \times W$

CBAM + ResNet Integration

Insertion Points

CBAM can be integrated at different positions in a residual block:

  • • After each convolution (most common)
  • • After the full bottleneck (before skip)
  • • Before the final output

Computational Overhead

Despite the added complexity, CBAM adds minimal overhead:

  • • Parameters: ~2C²/r (shared MLP)
  • • FLOPs: Negligible compared to conv layers
  • • Typically r=16 for good efficiency

PyTorch Implementation

import torch
import torch.nn as nn

class ChannelAttention(nn.Module):
    """Channel Attention Module"""
    def __init__(self, in_channels, reduction_ratio=16):
        super(ChannelAttention, self).__init__()
        self.avg_pool = nn.AdaptiveAvgPool2d(1)
        self.max_pool = nn.AdaptiveMaxPool2d(1)
        
        # Shared MLP
        self.mlp = nn.Sequential(
            nn.Conv2d(in_channels, in_channels // reduction_ratio, 1, bias=False),
            nn.ReLU(),
            nn.Conv2d(in_channels // reduction_ratio, in_channels, 1, bias=False)
        )
        self.sigmoid = nn.Sigmoid()
    
    def forward(self, x):
        # Generate attention weights
        avg_out = self.mlp(self.avg_pool(x))
        max_out = self.mlp(self.max_pool(x))
        
        # Combine and apply sigmoid
        channel_att = self.sigmoid(avg_out + max_out)
        return channel_att


class SpatialAttention(nn.Module):
    """Spatial Attention Module"""
    def __init__(self, kernel_size=7):
        super(SpatialAttention, self).__init__()
        self.conv = nn.Conv2d(2, 1, kernel_size, padding=kernel_size//2, bias=False)
        self.sigmoid = nn.Sigmoid()
    
    def forward(self, x):
        # Channel-wise pooling
        avg_out = torch.mean(x, dim=1, keepdim=True)
        max_out, _ = torch.max(x, dim=1, keepdim=True)
        
        # Concatenate and convolve
        spatial_input = torch.cat([avg_out, max_out], dim=1)
        spatial_att = self.sigmoid(self.conv(spatial_input))
        return spatial_att


class CBAM(nn.Module):
    """Convolutional Block Attention Module"""
    def __init__(self, in_channels, reduction_ratio=16, spatial_kernel=7):
        super(CBAM, self).__init__()
        self.channel_attention = ChannelAttention(in_channels, reduction_ratio)
        self.spatial_attention = SpatialAttention(spatial_kernel)
    
    def forward(self, x):
        # Apply channel attention
        x = x * self.channel_attention(x)
        
        # Apply spatial attention
        x = x * self.spatial_attention(x)
        return x

Key Insights & Impact

Why It Works

  • ✓ Dual attention: Both channel (what) and spatial (where) dimensions are considered
  • ✓ Sequential processing: Channel-first, then spatial provides better feature refinement
  • ✓ Multi-pooling: Avg and Max pooling provide complementary information
  • ✓ Lightweight: Minimal parameters and computation overhead

Research Impact

  • ★ Pioneered dual-branch attention mechanisms
  • ★ Influenced subsequent attention modules (ECA-Net, etc.)
  • ★ Demonstrated plug-and-play attention for any CNN
  • ★ Validated across classification, detection, and segmentation

Evolution of CBAM

2017 - SENet

Channel-only attention using global average pooling

2018 - CBAM

Sequential channel + spatial attention with multi-pooling

2020 - ECANet

Efficient channel attention via 1D convolution

2021+ - Self-Attention Era

Transformers dominate with multi-head self-attention
