AlphaGo Zero

Mastering the game of Go without human knowledge

The Problem

Before AlphaGo Zero, even the most advanced AI systems relied heavily on human knowledge:

  • Human Data Dependency: Previous AlphaGo versions required thousands of expert human games for supervised learning
  • Feature Engineering: Manual construction of input features based on human Go knowledge
  • Separate Architectures: Policy and value networks trained independently with different architectures
  • Complex Pipelines: Multiple training stages combining supervised and reinforcement learning
  • Computational Inefficiency: Required distributed training across many machines

Core Question: Can an AI achieve superhuman performance in the most challenging domains through pure reinforcement learning, starting from random play with no human data beyond the basic rules?

Key Innovation

AlphaGo Zero introduced a pure reinforcement learning paradigm that learns entirely through self-play, achieving superhuman performance tabula rasa (from a blank slate).

AlphaGo Zero Architecture

Architecture & Method

1. Unified Neural Network

AlphaGo Zero uses a single residual network that outputs both policy probabilities and value estimation:

(p, v) = fθ(s)

Where p is the probability vector over all possible moves, and v estimates the win probability from the current player's perspective.

2. Residual Network Architecture

  • Input: 19×19 board + 7 history boards + 1 color plane = 17 input channels
  • Conv Tower: Initial convolution with 256 filters
  • Residual Blocks: 19 blocks (small) or 39 blocks (large) with skip connections
  • Policy Head: 2 conv layers → 362 outputs (19×19 board + pass)
  • Value Head: 1 conv → 256 FC → tanh scalar output [-1, +1]

3. MCTS with Neural Network Guidance

Each simulation in MCTS uses the neural network to guide search:

PUCT: U(s,a) = cpuct × P(s,a) × √N(s) / (1 + N(s,a))

The search probabilities π are derived from visit counts: πa ∝ N(s,a)1/τ

4. Self-Play Training

  • Generate games by playing the current best network against itself
  • At each position, run 1,600 MCTS simulations to get search probabilities π
  • Sample moves from π with temperature parameter for exploration
  • Game outcome z = +1 for win, -1 for loss from current player's view
  • Store training examples (s, π, z) for mini-batch updates

Historic Achievement: AlphaGo Zero defeated AlphaGo Lee (the version that beat Lee Sedol 4-1) by 100 games to 0, after just 3 days of training.

Key Discoveries

  • Rediscovered Human Knowledge: Learned all known Go strategies and tactics developed over millennia
  • Novel Strategies: Discovered new opening moves and joseki patterns previously unknown to humans
  • Superior Efficiency: Trained on 4.9M games vs 30M games for AlphaGo Lee
  • Simpler Architecture: One network vs two separate networks, no rollouts
  • Pure Learning: No human bias - discovered optimal play through pure reasoning

Training Progression

AlphaGo Zero's learning curve showed remarkable progress:

  • 8 hours: Amateur human level
  • 36 hours: Professional player level
  • 72 hours (3 days): Superhuman - beat AlphaGo Lee 100-0
  • 21 days: Surpassed AlphaGo Master
  • 40 days: Peak performance - discovered all known Go knowledge

Code Implementation

# AlphaGo Zero-style Self-Play Training (PyTorch-style pseudocode) class AlphaGoZero: def __init__(self, board_size=19, num_res_blocks=19): self.network = ResNet(board_size, num_res_blocks) self.mcts = MCTS(self.network, c_puct=1.5, num_simulations=1600) def self_play(self, num_games=25000): # Generate training data through self-play training_data = [] for game in range(num_games): states, policies, current_player = [], [], [] board = GoBoard() while not board.is_terminal(): state = board.get_state() # 17 channels: current + 7 history + color # MCTS search with neural network guidance pi = self.mcts.search(board) # π from visit counts states.append(state) policies.append(pi) current_player.append(board.current_player) # Sample action from search probabilities action = np.random.choice(len(pi), p=pi) board.play(action) # Game outcome: z = +1 for win, -1 for loss winner = board.get_winner() for i, player in enumerate(current_player): z = 1 if player == winner else -1 training_data.append((states[i], policies[i], z)) return training_data def train(self, training_data, epochs=1000, batch_size=2048): optimizer = torch.optim.SGD(self.network.parameters(), lr=0.02, momentum=0.9, weight_decay=1e-4) scheduler = torch.optim.lr_scheduler.MultiStepLR( optimizer, milestones=[400, 600, 800], gamma=0.1) for epoch in range(epochs): for batch in DataLoader(training_data, batch_size): states, target_pi, target_z = batch # Forward pass: get policy and value predictions pred_p, pred_v = self.network(states) # Loss = MSE(value) + CE(policy) + L2_regularization value_loss = (target_z - pred_v).pow(2).mean() policy_loss = -(target_pi * torch.log(pred_p)).sum(dim=1).mean() loss = value_loss + policy_loss optimizer.zero_grad() loss.backward() optimizer.step() scheduler.step()

Why It Matters

AlphaGo Zero represents a paradigm shift in AI research - proving that superhuman performance can be achieved through pure learning, without human expertise. This has profound implications:

  • Tabula Rasa Learning: Demonstrated that AI can discover knowledge from first principles, not just mimic human experts
  • Generalizability: The same approach (later called AlphaZero) mastered Chess and Shogi without game-specific changes
  • Efficiency: Self-play with correct exploration is more data-efficient than human expert imitation
  • Pure RL Works: Validated that reinforcement learning alone, with sufficient computation and exploration, can solve complex problems
  • Inspiration for MuZero: Led to model-based RL that learns the environment dynamics

Legacy: AlphaGo Zero's approach of combining deep neural networks with planning (MCTS) and pure self-play has influenced not just game-playing AI, but also protein folding (AlphaFold), materials discovery, and optimization problems across science.
