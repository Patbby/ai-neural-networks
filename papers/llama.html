LLaMA: Open and Efficient Foundation Language Models

A collection of foundation language models ranging from 7B to 65B parameters, trained on trillions of tokens

📅 2023 👤 Meta AI (Touvron et al.) 📄 arXiv:2302.13971
📄 Paper (arXiv) 💻 Code 🌐 Blog

Problem Statement

Large language models (LLMs) have shown remarkable capabilities, but access to these models has been limited due to:

  • Closed-source nature of most capable models (GPT-3, PaLM, etc.)
  • Prohibitive computational costs for training and inference
  • Limited ability for researchers to study and improve these models
  • Unclear relationship between model size, training data, and performance

Existing open-source models like GPT-J, GPT-NeoX, and OPT were available but generally underperformed compared to their closed-source counterparts of similar sizes.

Key Innovation

LLaMA demonstrates that training smaller models on more data can achieve better performance than training larger models on less data. The core innovations include:

🎯 Pre-normalization (RMSNorm)

Normalize the input of each transformer sub-layer instead of the output, improving training stability. Uses RMSNorm instead of LayerNorm.

⚡ SwiGLU Activation

Replaces ReLU with SwiGLU activation function for improved performance. Uses 2/3 of the hidden dimension compared to PaLM.

🔄 Rotary Positional Embeddings (RoPE)

Removes absolute positional embeddings and adds rotary positional embeddings at each layer, providing better length generalization.

📊 Efficient Training

Trained on 1.0-1.4 trillion tokens using publicly available datasets only, with AdamW optimizer and cosine learning rate schedule.

Code Implementation

Here's a minimal PyTorch implementation of the key LLaMA components:

Python / PyTorch
import torch
import torch.nn as nn
import torch.nn.functional as F
import math

class RMSNorm(nn.Module):
    """Root Mean Square Layer Normalization"""
    def __init__(self, dim: int, eps: float = 1e-6):
        super().__init__()
        self.eps = eps
        self.weight = nn.Parameter(torch.ones(dim))
    
    def forward(self, x):
        norm = x * torch.rsqrt(x.pow(2).mean(-1, keepdim=True) + self.eps)
        return norm * self.weight

class RotaryPositionalEmbedding(nn.Module):
    """Rotary Position Embedding (RoPE)"""
    def __init__(self, dim: int, max_seq_len: int = 2048):
        super().__init__()
        inv_freq = 1.0 / (10000 ** (torch.arange(0, dim, 2).float() / dim))
        self.register_buffer("inv_freq", inv_freq)
    
    def forward(self, seq_len: int, device):
        t = torch.arange(seq_len, device=device).type_as(self.inv_freq)
        freqs = torch.einsum("i,j->ij", t, self.inv_freq)
        emb = torch.cat((freqs, freqs), dim=-1)
        return emb.cos(), emb.sin()

def rotate_half(x):
    x1, x2 = x.chunk(2, dim=-1)
    return torch.cat((-x2, x1), dim=-1)

def apply_rotary_pos_emb(q, k, cos, sin):
    return (q * cos) + (rotate_half(q) * sin), (k * cos) + (rotate_half(k) * sin)

class SwiGLU(nn.Module):
    """SwiGLU Activation Function"""
    def __init__(self, dim: int, hidden_dim: int = None):
        super().__init__()
        if hidden_dim is None:
            hidden_dim = int(2/3 * 4 * dim)  # LLaMA uses 2/3 of 4d
        self.w1 = nn.Linear(dim, hidden_dim, bias=False)
        self.w2 = nn.Linear(hidden_dim, dim, bias=False)
        self.w3 = nn.Linear(dim, hidden_dim, bias=False)
    
    def forward(self, x):
        return self.w2(F.silu(self.w1(x)) * self.w3(x))

class LLaMAAttention(nn.Module):
    """Multi-head attention with RoPE"""
    def __init__(self, dim: int, n_heads: int):
        super().__init__()
        self.n_heads = n_heads
        self.head_dim = dim // n_heads
        
        self.wq = nn.Linear(dim, dim, bias=False)
        self.wk = nn.Linear(dim, dim, bias=False)
        self.wv = nn.Linear(dim, dim, bias=False)
        self.wo = nn.Linear(dim, dim, bias=False)
        
        self.rope = RotaryPositionalEmbedding(self.head_dim)
    
    def forward(self, x, mask=None):
        bsz, seqlen, _ = x.shape
        
        q, k, v = self.wq(x), self.wk(x), self.wv(x)
        q = q.view(bsz, seqlen, self.n_heads, self.head_dim).transpose(1, 2)
        k = k.view(bsz, seqlen, self.n_heads, self.head_dim).transpose(1, 2)
        v = v.view(bsz, seqlen, self.n_heads, self.head_dim).transpose(1, 2)
        
        # Apply RoPE
        cos, sin = self.rope(seqlen, x.device)
        cos, sin = cos.unsqueeze(0).unsqueeze(0), sin.unsqueeze(0).unsqueeze(0)
        q, k = apply_rotary_pos_emb(q, k, cos, sin)
        
        # Attention
        scores = torch.matmul(q, k.transpose(-2, -1)) / math.sqrt(self.head_dim)
        if mask is not None:
            scores = scores.masked_fill(mask == 0, float('-inf'))
        attn = F.softmax(scores, dim=-1)
        
        out = torch.matmul(attn, v)
        out = out.transpose(1, 2).contiguous().view(bsz, seqlen, -1)
        return self.wo(out)

class LLaMATransformerBlock(nn.Module):
    """Single LLaMA transformer block"""
    def __init__(self, dim: int, n_heads: int):
        super().__init__()
        self.attention_norm = RMSNorm(dim)
        self.attention = LLaMAAttention(dim, n_heads)
        self.ffn_norm = RMSNorm(dim)
        self.ffn = SwiGLU(dim)
    
    def forward(self, x, mask=None):
        # Pre-normalization + Attention with residual
        h = x + self.attention(self.attention_norm(x), mask)
        # Pre-normalization + FFN with residual
        out = h + self.ffn(self.ffn_norm(h))
        return out

Legacy & Impact

LLaMA's release marked a pivotal moment in democratizing access to powerful language models:

  • Open Source Movement: Sparked the open-source LLM revolution, leading to Alpaca, Vicuna, and countless other fine-tuned variants
  • Research Accessibility: Enabled researchers worldwide to study and improve large language models
  • Training Efficiency Insights: Demonstrated the importance of training data scale vs. model size
  • Commercial Adoption: Led to LLaMA 2 and LLaMA 3 with commercial licenses
  • Community Innovation: Inspired quantization techniques (GGML, GPTQ) for running LLaMA on consumer hardware

LLaMA proved that smaller, efficiently trained models can match or exceed the performance of larger, proprietary models—fundamentally changing how the AI community approaches language model development.
