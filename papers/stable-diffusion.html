🎯 Problem Statement

Diffusion Models (DMs) achieved state-of-the-art synthesis results on image generation, but they came with a critical limitation: they operated directly in pixel space. This made them extremely resource-intensive:

Training: Hundreds of GPU days | Inference: Sequential & Expensive
Pixel-space DMs require evaluating gradients and neural networks on every single pixel

The Computational Bottleneck

Most bits in a digital image correspond to imperceptible high-frequency details — information that is semantically meaningless but computationally expensive to process. Pixel-space diffusion models must evaluate on all pixels, leading to:

❌ Superfluous Computation
Processing imperceptible noise

❌ Expensive Training
Hundreds of GPU days required

❌ Slow Inference
Sequential denoising steps

❌ Limited Accessibility
Only viable for large organizations

Key Questions

❓ Can we separate perceptual compression from semantic generation?
❓ Is there a perceptually equivalent but computationally more suitable space?
❓ How can we enable high-resolution synthesis on consumer hardware?

💡 Key Innovation: Latent Diffusion

Latent Diffusion Models (LDMs) decompose image synthesis into two distinct stages:

Two-Stage Learning Process

Why This Works

1. Perceptual Compression: A pretrained autoencoder compresses images into a lower-dimensional latent space, removing imperceptible details while preserving semantic content.

2. Semantic Compression: The diffusion model operates in this efficient latent space, learning the semantic and conceptual composition of data.

3. Cross-Attention Conditioning: General-purpose conditioning mechanism enables text-to-image, class-conditional, and layout-to-image synthesis through cross-attention layers.

LLDM = Ez,ε~N(0,1),t[||ε - εθ(zt, t, τθ(y))||²₂]
Latent diffusion loss: predict noise ε given noisy latent zt, timestep t, and conditioning y

import torch
import torch.nn as nn

class LatentDiffusion(nn.Module):
    """Latent Diffusion Model with Cross-Attention Conditioning"""
    
    def __init__(self, unet, autoencoder, text_encoder):
        super().__init__()
        self.unet = unet              # Denoising UNet
        self.vae = autoencoder        # AutoencoderKL or VQGAN
        self.text_enc = text_encoder  # CLIP/BERT text encoder
        
    def encode(self, x):
        """Encode image to latent space"""
        return self.vae.encode(x).sample()
    
    def decode(self, z):
        """Decode latent to image"""
        return self.vae.decode(z)
    
    def forward(self, x, text, t):
        # Encode image to latent
        z = self.encode(x)
        
        # Add noise (forward diffusion)
        noise = torch.randn_like(z)
        z_t = self.q_sample(z, t, noise)
        
        # Encode text conditioning
        c = self.text_enc(text)
        
        # Predict noise with cross-attention
        noise_pred = self.unet(z_t, t, context=c)
        
        # MSE loss between predicted and actual noise
        loss = F.mse_loss(noise_pred, noise)
        return loss
    
    def q_sample(self, z, t, noise):
        """Forward diffusion: z_t = √(ᾱ_t) z₀ + √(1-ᾱ_t) ε"""
        sqrt_alpha_t = self.sqrt_alphas_cumprod[t]
        sqrt_one_minus_alpha_t = self.sqrt_one_minus_alphas_cumprod[t]
        return sqrt_alpha_t * z + sqrt_one_minus_alpha_t * noise


class CrossAttention(nn.Module):
    """Cross-attention for conditioning on text/image features"""
    
    def __init__(self, query_dim, context_dim, heads=8):
        super().__init__()
        self.heads = heads
        self.scale = (query_dim // heads) ** -0.5
        
        self.to_q = nn.Linear(query_dim, query_dim)
        self.to_k = nn.Linear(context_dim, query_dim)
        self.to_v = nn.Linear(context_dim, query_dim)
        self.to_out = nn.Linear(query_dim, query_dim)
    
    def forward(self, x, context):
        # x: [batch, seq_len, dim], context: [batch, ctx_len, ctx_dim]
        q = self.to_q(x)
        k = self.to_k(context)
        v = self.to_v(context)
        
        # Multi-head attention
        q, k, v = [self.reshape_heads(t) for t in (q, k, v)]
        
        attn = torch.softmax(q @ k.transpose(-2, -1) * self.scale, dim=-1)
        out = attn @ v
        
        return self.to_out(self.reshape_batch(out))

Key Training Details

• Autoencoder: KL-regularized VAE with f=4 or f=8 downsampling factor

• UNet: ResNet backbone with self-attention at 16×16 resolution

• Timesteps: T=1000 for training, DDIM sampler for efficient inference

• Conditioning: Cross-attention layers at multiple resolutions

• Optimizer: AdamW, learning rate 1e-4
