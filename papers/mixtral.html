📋 Abstract

Mixtral 8x7B is a Sparse Mixture of Experts (SMoE) language model that outperforms Llama 2 70B on most benchmarks while using only 12.9B active parameters per token — matching a 13B dense model's inference cost despite having 46.7B total parameters.

"Mixtral matches or outperforms Llama 2 70B and GPT-3.5 on most benchmarks. On coding and math, it significantly outperforms — all with 6× faster inference."

⚙️ Sparse MoE Architecture

In each transformer layer, Mixtral replaces the standard FFN with 8 expert FFN networks. A router network selects the top-2 experts for each token:

MoE Layer: Top-2 Routing per Token

Only 2 out of 8 experts are computed per token — achieving 46.7B parameter expressiveness at the cost of ~13B parameters per forward pass.

🔑 Architecture Details

  • Sliding Window Attention: Each layer attends to W previous tokens; information propagates through depth — enables 32K context efficiently
  • Rolling KV Cache: Fixed-size cache that recycles old keys/values, compatible with sliding window
  • Grouped Query Attention: Faster inference with fewer KV heads
  • Expert routing formula: softmax(TopK(x·W_g)) — router is a simple linear layer
  • Load balancing: Auxiliary loss encourages equal expert utilization during training

💡 Impact & Key Takeaways

  • Proved SMoE can match much larger dense models at inference cost of a small model
  • Context length of 32K tokens — 4× longer than Llama 2
  • Enabled Mixtral-8×22B and Mixtral Instruct variants widely used in production
  • Inspired subsequent MoE architectures: Deepseek-MoE, Qwen MoE, DBRX
  • Released fully openly — immediately deployable on consumer hardware (quantized)
