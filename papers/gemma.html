Gemma

Gemma is a family of lightweight, open-weight language models from Google DeepMind, built on the same research and infrastructure as Gemini. Available in 2B and 7B parameter sizes with state-of-the-art performance per parameter count.

Organization: Google DeepMind
Released: Feb 2024
Sizes: 2B / 7B
License: Gemma Terms (Open)

Architecture Overview

Gemma uses a decoder-only Transformer architecture with several key improvements over standard GPT-style models, including Multi-Query Attention (MQA), RoPE positional embeddings, and GeGLU activations. The architecture is directly derived from Gemini's design principles, optimized for efficient deployment.

Key Design Choices

🔑

Multi-Query Attention

Shares Key-Value heads across all Query heads, dramatically reducing KV-cache memory usage at inference — critical for deployment on consumer hardware.

🌀

RoPE Positional Encoding

Rotary Position Embeddings applied to each attention layer instead of learned absolute positional embeddings, enabling better length generalization.

⚡

GeGLU Activation

Gated Linear Unit with GELU inside the FFN layers, providing better gradient flow and improved performance over standard ReLU/GELU activations.

📏

RMSNorm (No Bias)

Pre-normalization with RMSNorm and no bias parameters throughout, following LLaMA's finding that bias-free models train more stably at scale.

Training Details

🔑 Responsible Deployment Philosophy

Unlike Llama (Meta's internal model released publicly), Gemma was designed from the start as an open-weights community model with responsible AI principles. Google applied RLHF and safety tuning comparable to their production models, then published model weights and technical details for the research community.

Impact & Significance

Gemma established that open-weight models from major labs can match or exceed independently-developed models at the same scale. The 7B model outperformed Llama-2 13B on most benchmarks despite having fewer parameters, demonstrating the value of Gemini-class training data and infrastructure applied to smaller models.

Gemma's release accelerated the open-weight LLM ecosystem and influenced subsequent releases including Gemma 2, Phi-3, and Llama 3. The responsible deployment approach — including detailed model cards, safety evaluations, and usage policies — set a new standard for open model releases from industry labs.
