🎯 Problem Statement

Before DiT, diffusion models for image generation almost exclusively used U-Net architectures as their backbone. The U-Net inductive bias (spatial downsampling/upsampling with skip connections) was considered essential for good performance.

However, this architectural choice meant that diffusion models couldn't benefit from the standardization, best practices, and scaling properties that transformers had demonstrated across other domains like NLP and vision recognition.

Key Question

Is the U-Net inductive bias actually crucial for diffusion model performance? Can we replace it with standard transformer architectures?

💡 Key Innovation

DiT demonstrates that transformers can directly replace U-Nets as the backbone of diffusion models with no loss in performance—and in fact, with better scaling properties.

The authors train latent diffusion models where the U-Net is replaced by a transformer operating on patches of latent tokens. They find a strong correlation between compute (Gflops) and sample quality (FID).

Core Insight

Diffusion models are well-poised to benefit from architecture unification. By using standard transformers, they can inherit scalability, robustness, and efficiency best practices from other domains.

📝 Key Results

  • ImageNet 256×256: FID 2.27 (previous SOTA: LDM-4 with FID 3.60)
  • ImageNet 512×512: FID 3.04 (previous SOTA: ADM with FID 3.85)
  • DiT achieves SOTA with significantly fewer Gflops than U-Net alternatives
  • Scaling model size is more efficient than increasing sampling steps

# DiT Block with Adaptive Layer Norm (adaLN)
import torch
import torch.nn as nn

class DiTBlock(nn.Module):
    def __init__(self, hidden_size, num_heads, mlp_ratio=4.0):
        super().__init__()
        self.norm1 = nn.LayerNorm(hidden_size, elementwise_affine=False)
        self.attn = nn.MultiheadAttention(hidden_size, num_heads)
        self.norm2 = nn.LayerNorm(hidden_size, elementwise_affine=False)
        mlp_hidden_dim = int(hidden_size * mlp_ratio)
        self.mlp = nn.Sequential(
            nn.Linear(hidden_size, mlp_hidden_dim),
            nn.GELU(),
            nn.Linear(mlp_hidden_dim, hidden_size)
        )
        # adaLN modulation: predicts γ, β, α from t and c
        self.adaLN_modulation = nn.Sequential(
            nn.SiLU(),
            nn.Linear(hidden_size, 6 * hidden_size, bias=True)
        )
    
    def forward(self, x, c):
        # c: conditioning embedding (timestep + class)
        shift_msa, scale_msa, gate_msa, shift_mlp, scale_mlp, gate_mlp = \
            self.adaLN_modulation(c).chunk(6, dim=-1)
        
        # Self-attention with adaLN
        x = x + gate_msa * self.attn(
            modulate(self.norm1(x), shift_msa, scale_msa)
        )
        
        # MLP with adaLN
        x = x + gate_mlp * self.mlp(
            modulate(self.norm2(x), shift_mlp, scale_mlp)
        )
        return x

def modulate(x, shift, scale):
    return x * (1 + scale) + shift

🌟 Impact & Influence

DiT has become the foundation for next-generation generative models:

  • Stable Diffusion 3: Adopted transformer-based architecture inspired by DiT
