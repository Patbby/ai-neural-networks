🎯 Problem Statement

Before DiT, diffusion models for image generation predominantly used U-Net architectures as their backbone. While effective, several limitations existed:

The U-Net Bottleneck

U-Nets rely on convolutional operations and inductive biases like translation equivariance. While these properties are beneficial, they may not be optimal for the diffusion process, which fundamentally operates on spatial representations that could benefit from global attention.

  • Limited Receptive Field: Convolutional layers have limited receptive fields, requiring deep stacks for global context
  • Inductive Bias Constraints: Built-in assumptions about spatial structure may limit the model's capacity
  • Scaling Challenges: U-Nets don't scale as predictably with increased compute (Gflops)
  • Redundant Computations: Skip connections and encoder-decoder structure introduce computational overhead

The Question: Can we replace the U-Net backbone with a more scalable architecture while maintaining or improving generation quality?

💡 Key Innovation

Replacing U-Net with Vision Transformer

DiT replaces the U-Net backbone in Latent Diffusion Models with a pure transformer that operates on latent patches. This simple change unlocks unprecedented scalability and achieves state-of-the-art image generation quality.

Core Ideas

  • Latent Patch Processing: The transformer operates on patches from the VAE latent space, not raw pixels
  • Standard Transformer Blocks: Uses standard Vision Transformer architecture with multi-head self-attention and FFN
  • AdaLN-Zero Conditioning: Uses adaptive layer normalization with zero initialization for stable conditioning
  • Predictable Scaling: Higher Gflops consistently lead to better FID scores

Key Finding: Predictable Scaling

DiT demonstrates that diffusion transformers scale predictably with increased computational budget (Gflops). Through increased depth, width, or input tokens, models with higher Gflops consistently achieve lower FID scores.

💻 Code Implementation

PyTorch DiT Block

import torch
import torch.nn as nn

class DiTBlock(nn.Module):
    """
    A DiT block with adaptive layer norm zero (adaLN-Zero) conditioning.
    """
    def __init__(self, hidden_size, num_heads, mlp_ratio=4.0):
        super().__init__()
        self.norm1 = nn.LayerNorm(hidden_size, elementwise_affine=False, eps=1e-6)
        self.attn = nn.MultiheadAttention(hidden_size, num_heads, batch_first=True)
        self.norm2 = nn.LayerNorm(hidden_size, elementwise_affine=False, eps=1e-6)
        mlp_hidden_dim = int(hidden_size * mlp_ratio)
        self.mlp = nn.Sequential(
            nn.Linear(hidden_size, mlp_hidden_dim),
            nn.GELU(),
            nn.Linear(mlp_hidden_dim, hidden_size)
        )
        
        # AdaLN modulation: produces scale, shift, gate for both attn and mlp
        self.adaLN_modulation = nn.Sequential(
            nn.SiLU(),
            nn.Linear(hidden_size, 6 * hidden_size, bias=True)
        )
        
        # Initialize modulation to zero
        nn.init.zeros_(self.adaLN_modulation[-1].weight)
        nn.init.zeros_(self.adaLN_modulation[-1].bias)

    def forward(self, x, c):
        # x: (B, N, D), c: (B, D) conditioning from timestep + label
        shift_msa, scale_msa, gate_msa, shift_mlp, scale_mlp, gate_mlp = \
            self.adaLN_modulation(c).chunk(6, dim=-1)
        
        # Attention block with adaLN
        x_norm = self.norm1(x)
        x_modulated = x_norm * (1 + scale_msa.unsqueeze(1)) + shift_msa.unsqueeze(1)
        attn_out, _ = self.attn(x_modulated, x_modulated, x_modulated)
        x = x + gate_msa.unsqueeze(1) * attn_out
        
        # MLP block with adaLN
        x_norm = self.norm2(x)
        x_modulated = x_norm * (1 + scale_mlp.unsqueeze(1)) + shift_mlp.unsqueeze(1)
        x = x + gate_mlp.unsqueeze(1) * self.mlp(x_modulated)
        
        return x

DiT Forward Pass

class DiT(nn.Module):
    def forward(self, x, t, y):
        # x: (B, C, H, W) latent from VAE
        # t: (B,) diffusion timestep
        # y: (B,) class labels
        
        # 1. Patchify: (B, C, H, W) -> (B, N, patch_dim)
        x = self.patchify(x)
        
        # 2. Add positional embeddings
        x = x + self.pos_embed
        
        # 3. Combine timestep and class conditioning
        t_emb = self.timestep_embedder(t)
        y_emb = self.label_embedder(y)
        c = t_emb + y_emb
        
        # 4. Pass through transformer blocks
        for block in self.blocks:
            x = block(x, c)
        
        # 5. Final norm and head
        x = self.final_adaLN(x, c)
        x = self.head(x)
        
        # 6. Unpatchify: (B, N, patch_dim) -> (B, C, H, W)
        x = self.unpatchify(x)
        
        return x

Key Insights from DiT

Architecture Choice Matters

DiT proved that the backbone architecture choice significantly impacts diffusion model performance. While U-Nets served well, transformers offer better scalability and achieved state-of-the-art results when scaled appropriately.
