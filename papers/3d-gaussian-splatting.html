Replace neural networks with explicit 3D Gaussian primitives to achieve real-time photorealistic rendering while maintaining quality competitive with Neural Radiance Fields (NeRF).

🏗️ Architecture & Method

Core Representation

The scene is represented as a collection of 3D anisotropic Gaussians, each defined by:

μ Center Position (x, y, z) in world coordinates
Σ Covariance Matrix 3×3 anisotropic shape
α Opacity Per-Gaussian transparency
SH Spherical Harmonics View-dependent color (0-3 degrees)

3D Gaussian Primitive

Covariance Decomposition

To ensure positive semi-definite covariance during optimization, the covariance matrix is decomposed into scaling and rotation:

Σ = RSSTRT

Where S is a diagonal scaling matrix and R is a rotation matrix (represented as quaternions for optimization stability).

🎨 Differentiable Rasterization Pipeline

1

View Transform

Project 3D Gaussians to 2D image space using EWA (Elliptical Weighted Average) splatting

2

Tile Sorting

Sort Gaussians by depth per 16×16 pixel tile using GPU-based radix sort

3

α-Blending

Accumulate colors front-to-back using standard alpha compositing

Gaussian Splatting Process

Projection Mathematics

The 3D covariance Σ is projected to 2D using the Jacobian of the affine camera transformation:

Σ′ = JW Σ WTJT

Where W is the viewing transformation and J is the Jacobian of the projective transformation.

⚙️ Optimization Strategy

Adaptive Density Control

The key innovation enabling high-quality reconstruction is adaptive Gaussian management during training:

📈 Densification

Split large Gaussians with high view-space gradient:

  • Trigger: ‖∇L‖ > threshold
  • Split into two smaller Gaussians
  • New scale: 1.6× original

🗑️ Pruning

Remove low-contribution Gaussians:

  • α < 0.005 (nearly transparent)
  • Too large in world space
  • Keep ~80% after warm-up

🆕 Cloning

Duplicate Gaussians in under-reconstructed regions:

  • Small Gaussians with high gradient
  • Copy to same position
  • Refine during optimization

Training Process

Loss Function

L = (1 - λ)L1 + λLSSIM

Combined L1 and SSIM loss with λ = 0.2. SSIM captures structural similarity while L1 preserves details.

Gaussian Representation (PyTorch)

import torch
import torch.nn as nn

class GaussianModel(nn.Module):
    def __init__(self, num_points=100_000):
        super().__init__()
        
        # Learnable parameters for each Gaussian
        self._xyz = nn.Parameter(torch.randn(num_points, 3) * 0.1)
        self._features_dc = nn.Parameter(torch.rand(num_points, 3))
        self._features_rest = nn.Parameter(torch.zeros(num_points, 45))  # SH
        self._opacity = nn.Parameter(torch.randn(num_points, 1))
        self._scaling = nn.Parameter(torch.randn(num_points, 3))
        self._rotation = nn.Parameter(torch.randn(num_points, 4))  # Quaternion
        
    def get_covariance(self, scaling_modifier=1.0):
        """Build 3x3 covariance from scale and rotation"""
        # Rotation from quaternion
        r = self._rotation  # [N, 4]
        R = build_rotation(r)  # [N, 3, 3]
        
        # Scale matrix
        S = torch.diag_embed(
            self._scaling * scaling_modifier
        )  # [N, 3, 3]
        
        # Covariance: RSS^T R^T
        covariance = R @ S @ S.transpose(-2, -1) @ R.transpose(-2, -1)
        return covariance
    
    def get_opacity(self):
        return torch.sigmoid(self._opacity)
    
    def get_features(self, view_direction):
        """Compute view-dependent color from SH"""
        # Evaluate spherical harmonics
        sh_features = self._features_dc + \
                      eval_sh(3, self._features_rest, view_direction)
        return torch.sigmoid(sh_features)

Rasterization (CUDA pseudo-code)

// Tile-based rasterization kernel
template <typename T>
__global__ void renderCUDA(
    const Gaussian2D* gaussians,  // Projected 2D Gaussians
    const float2* points_xy,
    const float4* conic_opacity,
    const float* colors,
    const uint2* tiles,           // Tile ranges per Gaussian
    uint32_t* contributions,        // Per-tile Gaussian indices
    float* final_color,
    float* final_depth
) {
    // Identify pixel
    uint32_t pix_x = blockIdx.x * blockDim.x + threadIdx.x;
    uint32_t pix_y = blockIdx.y * blockDim.y + threadIdx.y;
    
    // Get tile range
    uint2 tile_range = tiles[blockIdx.z];
    
    // Alpha blending loop
    float T = 1.0f;  // Transmittance
    float3 color = {0, 0, 0};
    float depth = 0;
    
    for (int i = tile_range.x; i < tile_range.y && T > 0.01f; i++) {
        // Load Gaussian
        Gaussian2D g = gaussians[contributions[i]];
        
        // Compute 2D Gaussian value
        float2 d = {pix_x - g.x, pix_y - g.y};
        float power = -0.5f * (conic_opacity[i].x * d.x * d.x + 
                               conic_opacity[i].z * d.y * d.y +
                               2 * conic_opacity[i].y * d.x * d.y);
        
        if (power > 0.0f) continue;
        
        float alpha = min(0.99f, conic_opacity[i].w * exp(power));
        float test_T = T * (1 - alpha);
        
        // Accumulate
        color += colors[i] * alpha * T;
        depth += g.depth * alpha * T;
        T = test_T;
    }
    
    // Write output
    final_color[pix_idx] = color;
}

Adaptive Control

def densify_and_split(self, grads, grad_threshold, scene_extent):
    """Split large Gaussians with high gradient"""
    # Find candidates
    big_gaussians = self.get_scaling().max(dim=1).values > scene_extent * 0.01
    high_grad = grads.norm(dim=-1) > grad_threshold
    
    candidates = big_gaussians & high_grad
    n_split = candidates.sum()
    
    if n_split == 0:
        return
    
    # Split: create two children per parent
    stds = self.get_scaling()[candidates].repeat(2, 1)
    means = torch.zeros((stds.size(0), 3), device="cuda")
    
    # Sample new positions around parent
    samples = torch.normal(mean=means, std=stds)
    
    # Update parameters
    new_xyz = self._xyz[candidates].repeat(2, 1) + samples
    new_scaling = self.scaling_inverse_activation(
        self.get_scaling()[candidates].repeat(2, 1) / (1.6)
    )
    
    # Update optimizer states
    self._xyz = nn.Parameter(torch.cat([self._xyz, new_xyz], dim=0))
    self._scaling = nn.Parameter(torch.cat([self._scaling, new_scaling], dim=0))
