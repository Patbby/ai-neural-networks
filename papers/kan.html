KAN: Kolmogorov-Arnold Networks

Replacing fixed activation functions with learnable B-spline activation functions on edges

📅 ICLR 2025 👤 Ziming Liu et al. (MIT, Caltech, Microsoft) 🏢 MIT, Caltech, Microsoft Research
📄 arXiv:2404.19756 📑 PDF 💻 PyKAN Code

🎯 The Problem: Fixed vs Learnable Nonlinearities

Multi-Layer Perceptrons (MLPs) have been the workhorse of deep learning for decades. Their architecture is deceptively simple: linear transformations followed by fixed, nonlinear activation functions (ReLU, GELU, SiLU, etc.). But this design has a fundamental limitation:

The MLP Paradigm: MLPs learn linear weights on edges and apply fixed nonlinearities on nodes. The nonlinearity (e.g., ReLU) is predetermined and cannot adapt to the data or task.

Kolmogorov-Arnold Networks (KANs) flip this paradigm on its head. Inspired by the Kolmogorov-Arnold representation theorem from mathematical analysis, KANs place learnable activation functions on edges instead of nodes.

Figure 1: MLP vs KAN Architecture

💡 The Innovation: Kolmogorov-Arnold Theorem

The Kolmogorov-Arnold representation theorem (1957) is a profound result in mathematical analysis:

Kolmogorov-Arnold Theorem: Any continuous multivariate function f: [0,1]ⁿ → ℝ can be represented as a finite composition of continuous univariate functions and addition:

f(x₁,...,xₙ) = Σᵢ Φᵢ(Σⱼ φᵢⱼ(xⱼ))

KANs operationalize this theorem directly in neural network architecture. Each edge in a KAN implements a learnable univariate function φ(x), parameterized as a B-spline:

KAN Edge Function:
φ(x) = w_b · b(x) + w_s · spline(x)

where b(x) = SiLU(x) = x/(1+e⁻ˣ) is the basis function, and spline(x) = Σᵢ cᵢBᵢ(x) is the learnable B-spline component.
Figure 2: KAN Edge Function - Learnable B-Spline Activation

🔧 Grid Extension: Progressive Refinement

A key innovation in KANs is grid extension—the ability to progressively refine the spline grid during training:

Coarse-to-Fine Training: Start with a coarse grid (few grid points), train the model, then extend to a finer grid with more points while preserving the learned function. This enables more efficient training and better scaling.

The grid extension uses the property that a finer spline can exactly represent a coarser spline. When extending from G intervals to 2G intervals:

  1. Interpolate coefficients for the new grid points
  2. Preserve the function values at all points
  3. Continue training with higher resolution

💻 PyTorch Implementation

Using KAN with the official PyKAN library:

import torch from kan import KAN # Create a KAN model # architecture: [2, 5, 1] means 2 inputs, 5 hidden, 1 output model = KAN(width=[2, 5, 1], grid=5, k=3) # width: network architecture # grid: number of grid intervals (G) # k: B-spline degree (typically 3 for cubic) # Training data x = torch.randn(100, 2) # 100 samples, 2 features y = torch.randn(100, 1) # 100 targets # Train the model model.train(x, y, opt="LBFGS", steps=100) # Make predictions y_pred = model(x) # Symbolic formula discovery! model.auto_symbolic() # Automatically discover symbolic form formula = model.symbolic_formula() print(f"Discovered formula: {formula}")

For grid extension during training:

# Train with coarse grid first model = KAN(width=[2, 5, 1], grid=3, k=3]) model.train(x, y, steps=50) # Extend to finer grid for better accuracy model = model.refine(10) # Extend from 3 to 10 grid intervals model.train(x, y, steps=50) # Can extend multiple times model = model.refine(20) # Further refinement

🔮 Interpretability: Rediscovering Physics

One of KANs' most compelling features is interpretability. The learned activation functions can reveal the underlying mathematical or physical laws:

Symbolic Recovery: KANs can "rediscover" known physics equations. For example, when trained on pendulum motion data, a KAN can learn to represent the exact formula for the period of a pendulum: T = 2π√(L/g)

This is possible because:

  • B-splines can approximate any continuous function arbitrarily well
  • The activation functions on edges can be pruned and simplified
  • Symbolic regression can extract formulas from learned activations
  • The network structure itself provides interpretable components
Example: When trained on data following Kepler's Third Law (T² ∝ a³), a KAN can learn activations that recover this exact relationship, while an MLP would learn a black-box approximation.

📝 Summary

Kolmogorov-Arnold Networks represent a paradigm shift from MLPs:

  • Learnable activations on edges: Instead of fixed σ on nodes, KANs learn univariate functions on edges
  • B-spline parameterization: Enables grid extension and progressive refinement
  • Better scaling laws: Smaller KANs can outperform larger MLPs
  • Interpretability: Can rediscover symbolic formulas and physical laws
  • Scientific applications: Particularly strong on PDE solving and function approximation
Bottom Line: KANs offer an intriguing alternative to MLPs, especially for scientific machine learning where interpretability matters. While current implementations are slower than MLPs, ongoing research (EfficientKAN, etc.) is addressing scalability. KANs demonstrate that neural network architecture still has room for fundamental innovation beyond the MLP paradigm that has dominated for decades.
