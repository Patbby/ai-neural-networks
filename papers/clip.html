Problem Statement

⚠️ The Limitation

State-of-the-art computer vision systems are trained to predict a fixed set of predetermined object categories. This restricted form of supervision limits their generality and usability since additional labeled data is needed to specify any other visual concept. Every new task requires collecting and annotating a new dataset.

Traditional vision models like ResNet trained on ImageNet can only recognize the 1,000 categories in the training set. To recognize new concepts, you need to collect images, annotate them, and retrain the model—an expensive and time-consuming process.

Key Innovation

💡 The Breakthrough

Learning directly from raw text about images leverages a much broader source of supervision. CLIP demonstrates that predicting which caption goes with which image is an efficient and scalable way to learn state-of-the-art image representations from 400 million (image, text) pairs collected from the internet.

After pre-training, natural language is used to reference learned visual concepts (or describe new ones), enabling zero-shot transfer to downstream tasks. You can classify images using natural language descriptions without any task-specific training data.

Training Objective

CLIP uses a contrastive learning objective across a batch of N (image, text) pairs:

  • Train the model to maximize the cosine similarity of N real pairs
  • Minimize the cosine similarity of N² - N incorrect pairings
  • Symmetric cross-entropy loss over similarity scores

CLIP achieved remarkable results without any ImageNet training data:

  • ImageNet: 76.2% top-1 accuracy zero-shot (matching ResNet-50 trained on 1.28M images)
  • Robustness: Better out-of-distribution performance than supervised models
  • Versatility: Works across OCR, action recognition, geo-localization, fine-grained classification
  • Efficiency: Same pre-trained model works for all downstream tasks

Code Implementation

Zero-Shot Image Classification with CLIP
import torch
import clip
from PIL import Image

# Load the model
device = "cuda" if torch.cuda.is_available() else "cpu"
model, preprocess = clip.load("ViT-B/32", device=device)

# Prepare image
image = preprocess(Image.open("image.jpg")).unsqueeze(0).to(device)

# Prepare text descriptions
text_descriptions = ["a photo of a cat", "a photo of a dog", "a photo of a bird"]
text_tokens = clip.tokenize(text_descriptions).to(device)

# Compute features
with torch.no_grad():
    image_features = model.encode_image(image)
    text_features = model.encode_text(text_tokens)
    
    # Normalize features
    image_features /= image_features.norm(dim=-1, keepdim=True)
    text_features /= text_features.norm(dim=-1, keepdim=True)
    
    # Compute similarity
    similarity = (100.0 * image_features @ text_features.T).softmax(dim=-1)
    
    # Get predictions
    probs = similarity[0].cpu().numpy()
    
# Print results
for label, prob in zip(text_descriptions, probs):
    print(f"{label}: {prob:.2%}")
