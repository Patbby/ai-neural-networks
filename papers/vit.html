An Image is Worth 16x16 Words:
Transformers for Image Recognition at Scale

📄 ICLR 2021 👤 Alexey Dosovitskiy, Lucas Beyer, Alexander Kolesnikov, et al. 🏢 Google Research
📑 arXiv 💻 Code 📄 ICLR 🔥 50,000+ Citations

🎯 核心问题

CNN在视觉领域的垄断地位：

在ViT之前，卷积神经网络(CNN)主导了计算机视觉领域超过8年(AlexNet 2012 - ViT 2020)。

Transformer在自然语言处理(NLP)领域取得了巨大成功，但将其应用于图像识别时，因为缺乏CNN的归纳偏置(局部性、平移不变性)，效果一直不如CNN。

核心挑战：如何将2D图像转换为Transformer可以处理的序列形式？

💡 解决方案：图像分块

ViT的创新： 将图像分割成固定大小的 patches (如 16×16 像素)，然后将每个 patch 展平并通过线性投影映射为向量，就像NLP中的"单词"(token)一样。

核心洞察： 只要数据足够大(如ImageNet-21k或JFT-300M)，纯Transformer架构可以不依赖CNN的归纳偏置，直接在图像分类任务上取得SOTA性能。

🏗️ ViT架构详解

Vision Transformer 完整流程

ViT将图像分块后通过线性投影嵌入，添加位置编码后输入Transformer编码器，仅使用[CLS]token进行分类

关键组件详解

z₀ = [xclass; x¹pE; x²pE; ...; xᴺpE] + Epos

Patch Embedding： 将每个 16×16 的图像块展平为 768 维向量 (16×16×3=768)。对于224×224的图像，得到 14×14=196 个patch。

Class Token： 与BERT类似，在序列开头添加一个可学习的 [CLS] token，其最终状态作为图像表示用于分类。

Position Embedding： 使用标准的1D可学习位置编码，给每个patch位置信息。

Self-Attention 在图像上的可视化

ViT的Self-Attention可以关注图像中任意位置的patch，学习全局依赖关系

💻 PyTorch 实现

import torch import torch.nn as nn import torchvision.models as models class PatchEmbedding(nn.Module): """图像分块并嵌入为向量""" def __init__(self, img_size=224, patch_size=16, in_chans=3, embed_dim=768): super().__init__() self.img_size = img_size self.patch_size = patch_size self.n_patches = (img_size // patch_size) ** 2 # 将patch展平并通过线性投影 self.proj = nn.Conv2d(in_chans, embed_dim, kernel_size=patch_size, stride=patch_size) def forward(self, x): # x: [B, 3, 224, 224] x = self.proj(x) # [B, 768, 14, 14] x = x.flatten(2) # [B, 768, 196] x = x.transpose(1, 2) # [B, 196, 768] return x class VisionTransformer(nn.Module): def __init__(self, img_size=224, patch_size=16, num_classes=1000, embed_dim=768, depth=12, num_heads=12, mlp_ratio=4): super().__init__() self.patch_embed = PatchEmbedding(img_size, patch_size, embed_dim=embed_dim) num_patches = self.patch_embed.n_patches # [CLS] token self.cls_token = nn.Parameter(torch.zeros(1, 1, embed_dim)) # 位置编码 self.pos_embed = nn.Parameter(torch.zeros(1, num_patches + 1, embed_dim)) # Transformer编码器 encoder_layer = nn.TransformerEncoderLayer( d_model=embed_dim, nhead=num_heads, dim_feedforward=mlp_ratio*embed_dim, dropout=0.1 ) self.transformer = nn.TransformerEncoder(encoder_layer, num_layers=depth) # 分类头 self.mlp_head = nn.Linear(embed_dim, num_classes) def forward(self, x): B = x.shape[0] x = self.patch_embed(x) # 添加[CLS] token cls_tokens = self.cls_token.expand(B, -1, -1) x = torch.cat([cls_tokens, x], dim=1) # 添加位置编码 x = x + self.pos_embed # Transformer x = self.transformer(x) # 使用[CLS] token进行分类 x = self.mlp_head(x[:, 0]) return x # 使用预训练模型vits = models.vit_b_16(pretrained=True) vitl = models.vit_l_16(pretrained=True)

🌍 影响与意义

ViT的出现标志着计算机视觉领域的范式转变：

统一架构： 证明了同一个架构(Transformer)可以同时在NLP和CV领域取得SOTA，推动了多模态研究的发展

数据优先： 当数据足够大时，架构的归纳偏置变得不那么重要

可扩展性： ViT展示了与NLP类似的scaling laws，更大的模型+更多数据=更好的性能

计算效率： 相比CNN，ViT在TPU/GPU上的并行训练效率更高

后续发展

  • DeiT (2020): 通过知识蒸馏，让小模型也能高效训练
  • Swin Transformer (2021): 引入层次化特征和移位窗口，结合CNN的局部性和Transformer的全局性
  • MAE (2021): 基于ViT的掩码自编码器，推动了视觉预训练的发展
  • DINO (2021): 自监督学习的视觉Transformer，涌现出类人的注意力模式
  • CLIP (2021): 使用ViT作为视觉编码器，实现视觉-语言联合学习
  • SAM (2023): Segment Anything Model，基于ViT的图像分割基础模型

📚 关键启示

ViT给我们的三个启示：

1. 统一架构： Transformer是通用的计算模块，不限于NLP

2. 数据规模： 足够的数据可以弥补归纳偏置的缺失

3. 灵活性： 自注意力机制提供了处理任意长度序列的灵活性
