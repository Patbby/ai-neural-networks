历史背景

1943
McCulloch & Pitts 提出人工神经元的数学模型，证明了神经网络可以计算任何可计算函数，为连接主义奠定基础。
1949
Hebb 提出"Hebbian 学习规则"——神经元同时激活时，它们之间的连接会增强。这成为生物学习机制的理论基础。
1957
Rosenblatt 在康奈尔航空实验室构建第一台感知机硬件 Mark I Perceptron，使用 400 个光电管作为输入，用于图像识别实验。
1958
发表里程碑论文《The Perceptron》，提出第一个具有学习能力的人工神经网络，开创了机器学习的全新范式。
1969
Minsky & Papert 出版《Perceptrons》，指出单层感知机无法解决非线性可分问题（如 XOR），导致神经网络研究陷入第一次寒冬。

问题陈述

🧠 大脑如何存储信息？

传统观点认为大脑像计算机一样存储编码后的符号表示。Rosenblatt 提出不同观点：信息存储在神经元之间的连接强度中，通过经验动态调整。

🎯 如何构建学习机器？

当时的人工智能研究主要依赖硬编码规则。Rosenblatt 希望创造一种能够像生物一样通过示例学习的机器，而非依赖人工编程。

核心洞察：大脑的学习能力来自于神经元连接的可塑性。如果一个机器能够调整其内部连接的权重，它就能够从数据中学习模式，而无需显式编程。

核心创新

  • 🎯

    感知机学习规则

    提出第一个可证明收敛的神经网络学习算法：如果分类错误，就调整权重使决策边界向正确方向移动。这是现代梯度下降的雏形。

  • 🔗

    连接主义范式

    确立了"连接主义"（Connectionism）的研究方向：智能行为可以从大量简单计算单元的连接和交互中涌现，而非符号操作。

  • 📊

    线性分类器

    将分类问题形式化为在高维空间中寻找分离超平面。输入的加权和通过阈值函数产生二值输出，开创了判别式模型的先河。

  • ⚙️

    硬件实现

    Mark I Perceptron 重达 5 吨，包含 400 个光电传感器、512 个电动电位器，能够识别字母和简单图形，证明了概念的可行性。

感知机架构

感知机是最简单的神经网络：输入层接收信号，输出层产生决策，通过可学习的权重连接。对于线性可分问题，它能找到最优决策边界。

📐 感知机收敛定理（Perceptron Convergence Theorem）

Rosenblatt 证明了：如果训练数据是线性可分的，感知机学习算法保证在有限步内收敛到完美分类器。

  • 初始化：权重 w = 0，偏置 b = 0
  • 预测：ŷ = sign(w·x + b)
  • 更新：如果 ŷ ≠ y，则 w ← w + η·y·x，b ← b + η·y
  • 收敛：经过有限次更新后，w 将完美分离两类数据

关键局限：1969年 Minsky & Papert 证明单层感知机无法解决非线性可分问题（如 XOR 问题）。这一发现导致神经网络研究进入第一次寒冬，直到多层网络和反向传播的出现才重新复兴。

PyTorch 实现

# 感知机 PyTorch 实现（带收敛性保证）
import torch
import torch.nn as nn

class Perceptron(nn.Module):
    """
    经典感知机实现
    1958年 Rosenblatt 提出的原始感知机算法
    """
    def __init__(self, input_dim):
        super(Perceptron, self).__init__()
        # 初始化权重为 0（与原始论文一致）
        self.weights = nn.Parameter(torch.zeros(input_dim))
        self.bias = nn.Parameter(torch.zeros(1))
    
    def forward(self, x):
        # 线性组合 + 符号函数
        linear = torch.matmul(x, self.weights) + self.bias
        return torch.sign(linear)
    
    def train_step(self, x, y, lr=1.0):
        # Rosenblatt 感知机学习规则
        prediction = self.forward(x)
        if prediction != y:
            # 错误时更新权重: w = w + lr * y * x
            self.weights.data += lr * y * x
            self.bias.data += lr * y
            return True  # 发生更新
        return False  # 未更新

# 训练示例：AND 问题（线性可分）
X = torch.tensor([[-1, -1], [-1, 1], 
                     [1, -1], [1, 1]], dtype=torch.float32)
y = torch.tensor([-1, -1, -1, 1], dtype=torch.float32)  # AND 标签

model = Perceptron(input_dim=2)
# 训练直到收敛
epochs = 0
while True:
    updates = 0
    for xi, yi in zip(X, y):
        if model.train_step(xi, yi):
            updates += 1
    epochs += 1
    if updates == 0:  # 收敛
        break

print(f"收敛于 {epochs} 轮，权重: {model.weights.data}")
