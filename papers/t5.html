T5

Text-to-Text Transfer Transformer | JMLR 2020

Colin Raffel, Noam Shazeer, Adam Roberts, et al. (Google)

📄 Paper 💻 Code 🏆 SOTA on GLUE & SuperGLUE

💡 Core Innovation

The Text-to-Text Paradigm: T5 treats every NLP task as a text-to-text problem — converting input text to output text using a unified framework with task-specific prefixes.
Text-to-Text Framework Visualization

The revolutionary insight of T5 is that every NLP task can be formulated as text generation. Whether it's translation, classification, summarization, or question answering — the model always produces text as output, eliminating the need for task-specific architectures.

🏗️ Architecture

T5 uses a standard encoder-decoder Transformer architecture, differentiating it from BERT (encoder-only) and GPT (decoder-only).

T5 Encoder-Decoder Architecture

Key Architectural Modifications

  • Pre-LayerNorm: Layer normalization is placed outside the residual path, unlike the original Transformer where it was inside
  • Simplified Relative Positional Embeddings: Uses a learnable scalar bias based only on relative distance between tokens, not sinusoidal embeddings
  • No LayerNorm Bias: Removes the additive bias from LayerNorm for improved training stability
  • GeLU Activation: Uses GeLU instead of ReLU in the feed-forward network
T5 LayerNorm:    y = (x / √mean(x²)) × weight

📚 Pre-training

C4 Dataset: Colossal Clean Crawled Corpus — 750GB of clean English text from Common Crawl, with quality filtering and deduplication.

Span Corruption Objective

Unlike BERT's masked language modeling (random token masking), T5 uses span corruption — masking consecutive spans of tokens and training the model to reconstruct them.

Span Corruption Example (15% masking, avg span length ~3)

Why Span Corruption?

Span corruption is more challenging than single-token masking and better aligns with the decoder's objective of generating sequences. It also more closely resembles real-world generation tasks like summarization and translation.

Scaling Performance (SuperGLUE Score)

The T5-11B model achieved 91.2 on SuperGLUE, surpassing the human baseline of 89.8 — a remarkable milestone showing that scale and unified transfer learning can push models beyond human performance on aggregated NLP benchmarks.

Key Contributions

  • Unified Text-to-Text Framework: Demonstrated that a single architecture with task prefixes can handle all NLP tasks, eliminating the need for custom output heads
  • Systematic Study of Transfer Learning: Comprehensive empirical analysis of what makes transfer learning work in NLP (architecture, pre-training objectives, datasets, fine-tuning strategies)
  • C4 Dataset: Released a massive, clean pre-training corpus that became a standard resource for NLP research
  • Scaling Laws: Demonstrated consistent performance improvements with model scale across diverse tasks

Evolution of the T5 Family

  • mT5 (2021): Multilingual version trained on 101 languages
  • ByT5 (2021): Token-free version operating directly on bytes
  • LongT5 (2022): Efficient modeling of long sequences
  • FLAN-T5 (2022): Instruction-tuned variant for better zero-shot performance
  • UL2 (2022): Mixture of denoisers objective combining multiple pre-training paradigms

Why T5 Matters

T5 showed that the text-to-text paradigm is a powerful unifying principle for NLP. By treating every task as text generation, T5 simplified the NLP pipeline and enabled a single model to excel across diverse tasks. This approach directly influenced later instruction-tuned models like FLAN, InstructGPT, and modern LLMs that use similar unified frameworks.
