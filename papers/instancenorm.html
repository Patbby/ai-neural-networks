Instance Normalization

The Missing Ingredient for Fast Stylization

👤 Dmitry Ulyanov, Andrea Vedaldi, Victor Lempitsky 📅 2016 📄 arXiv:1607.08022
📄 arXiv 💻 PyTorch

🎯 Problem Statement

Before Instance Normalization, neural style transfer faced a critical challenge: batch normalization was conflating instance-specific style information with batch statistics, making it impossible to generate stylized images efficiently in a feed-forward manner.

The Core Issue: Batch Normalization computes mean and variance across the entire batch, which normalizes away the very style differences that make each image unique. For style transfer, we need to preserve instance-specific characteristics while normalizing.

The authors discovered that simply replacing BatchNorm with InstanceNorm in the generator network enabled real-time neural style transfer — reducing generation time from iterative optimization (minutes) to a single forward pass (milliseconds).

💡 Key Innovation

Instance Normalization normalizes each individual sample (instance) independently, computing mean and variance across spatial dimensions for each channel:

IN(x) = γ × (x − μt) / √(σt² + ε) + β

Where for each instance t and channel k:

μtk = (1/HW) Σm=1..W Σn=1..H xtkmn
σtk² = (1/HW) Σm=1..W Σn=1..H (xtkmn − μtk)²

Key Insight: The authors discovered that normalizing each instance independently (rather than across a batch) preserves the relative style information while removing instance-specific contrast, enabling feed-forward style transfer.

💻 Code Implementation

PyTorch implementation of Instance Normalization:

import torch
import torch.nn as nn

# Built-in Instance Normalization
in_norm = nn.InstanceNorm2d(num_features=64, affine=True)

# Manual implementation
def instance_norm(x, eps=1e-5):
    # x: (N, C, H, W)
    mean = x.mean(dim=[2, 3], keepdim=True)  # Mean across H, W per instance
    var = x.var(dim=[2, 3], keepdim=True, unbiased=False)
    x_norm = (x - mean) / torch.sqrt(var + eps)
    return x_norm

# Style transfer generator example
class StyleTransferNet(nn.Module):
    def __init__(self):
        super().__init__()
        self.encoder = nn.Sequential(
            nn.Conv2d(3, 64, 3, padding=1),
            nn.InstanceNorm2d(64, affine=True),  # Key: IN instead of BN
            nn.ReLU(),
            nn.Conv2d(64, 128, 3, stride=2, padding=1),
            nn.InstanceNorm2d(128, affine=True),
            nn.ReLU(),
        )
        # ... decoder with IN layers
    
    def forward(self, x):
        return self.encoder(x)
