Overview

AlexNet marked a watershed moment in deep learning history. It was the first convolutional neural network to win the ImageNet Large Scale Visual Recognition Challenge (ILSVRC) in 2012, achieving a top-5 error rate of 15.3% — more than 10 percentage points better than the runner-up. This breakthrough demonstrated the power of deep learning and sparked the AI revolution that followed.

Architecture Details

AlexNet's architecture consists of 5 convolutional layers followed by 3 fully connected layers. The network was designed to run on two GPUs simultaneously, with half of the neurons on each GPU. This "two-stream" architecture was a necessity due to GPU memory constraints at the time but also provided a form of model parallelism.

Key Innovations

⚡
ReLU Activation
AlexNet was the first to use ReLU (Rectified Linear Unit) as the activation function instead of traditional saturating nonlinearities like tanh or sigmoid. ReLU trains 6x faster: f(x) = max(0, x)
💧
Dropout Regularization
Used dropout with probability 0.5 in the first two fully-connected layers to prevent overfitting. This randomly sets neuron outputs to zero during training.
🔄
Overlapping Pooling
Traditional pooling used non-overlapping neighborhoods. AlexNet used overlapping max pooling (stride smaller than kernel size), reducing top-1 and top-5 error rates.
🖼️
Data Augmentation
Random 224×224 patches cropped from 256×256 images, horizontal reflections, and PCA-based color augmentation to artificially enlarge the training set.
🖥️
GPU Training
First CNN to use GPUs for acceleration. The model was split across two GTX 580 GPUs with 3GB memory each, enabling training of a 60M parameter network.
📊
Local Response Normalization
Applied after ReLU in the first two convolutional layers to aid generalization by creating competition among neuron outputs computed using different kernels.

alexnet.py PyTorch 
import torch
import torch.nn as nn

class AlexNet(nn.Module):
    def __init__(self, num_classes=1000):
        super(AlexNet, self).__init__()
        
        # Feature extraction layers (5 convolutional layers)
        self.features = nn.Sequential(
            # Layer 1: Conv + ReLU + MaxPool + LRN
            nn.Conv2d(3, 96, kernel_size=11, stride=4, padding=2),
            nn.ReLU(inplace=True),
            nn.MaxPool2d(kernel_size=3, stride=2),
            nn.LocalResponseNorm(size=5, alpha=1e-4, beta=0.75, k=2),
            
            # Layer 2: Conv + ReLU + MaxPool + LRN
            nn.Conv2d(96, 256, kernel_size=5, padding=2),
            nn.ReLU(inplace=True),
            nn.MaxPool2d(kernel_size=3, stride=2),
            nn.LocalResponseNorm(size=5, alpha=1e-4, beta=0.75, k=2),
            
            # Layer 3: Conv + ReLU
            nn.Conv2d(256, 384, kernel_size=3, padding=1),
            nn.ReLU(inplace=True),
            
            # Layer 4: Conv + ReLU
            nn.Conv2d(384, 384, kernel_size=3, padding=1),
            nn.ReLU(inplace=True),
            
            # Layer 5: Conv + ReLU + MaxPool
            nn.Conv2d(384, 256, kernel_size=3, padding=1),
            nn.ReLU(inplace=True),
            nn.MaxPool2d(kernel_size=3, stride=2),
        )
        
        # Classification layers (3 fully connected layers)
        self.classifier = nn.Sequential(
            nn.Dropout(p=0.5),
            nn.Linear(256 * 6 * 6, 4096),
            nn.ReLU(inplace=True),
            nn.Dropout(p=0.5),
            nn.Linear(4096, 4096),
            nn.ReLU(inplace=True),
            nn.Linear(4096, num_classes),
        )
        
        # Initialize weights
        self._initialize_weights()
    
    def forward(self, x):
        x = self.features(x)
        x = torch.flatten(x, 1)
        x = self.classifier(x)
        return x
    
    def _initialize_weights(self):
        for m in self.modules():
            if isinstance(m, nn.Conv2d):
                nn.init.kaiming_normal_(m.weight, mode='fan_out', nonlinearity='relu')
                if m.bias is not None:
                    nn.init.constant_(m.bias, 0)
            elif isinstance(m, nn.Linear):
                nn.init.normal_(m.weight, 0, 0.01)
                nn.init.constant_(m.bias, 0)

Training Details

Optimization Configuration

AlexNet was trained using stochastic gradient descent (SGD) with the following hyperparameters:

  • Batch size: 128 (split across two GPUs)
  • Learning rate: 0.01, manually reduced when validation error plateaus
  • Momentum: 0.9
  • Weight decay: 0.0005 (L2 regularization)
  • Dropout ratio: 0.5 in fully connected layers

Training Time

The network was trained for approximately 90 epochs over the course of 5-6 days on two NVIDIA GTX 580 GPUs with 3GB of memory each. Training 60 million parameters required careful memory management and the two-GPU architecture was born out of necessity due to hardware limitations.

Historical Significance

AlexNet's victory in the 2012 ImageNet competition is widely regarded as the moment that sparked the deep learning revolution. The massive improvement over traditional computer vision methods demonstrated that:

  • Deep neural networks could learn hierarchical feature representations directly from raw pixels
  • GPUs were essential for training large-scale neural networks
  • Techniques like ReLU, dropout, and data augmentation were crucial for practical deep learning
