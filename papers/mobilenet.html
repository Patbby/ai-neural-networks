MobileNet

Efficient Convolutional Neural Networks for Mobile Vision Applications

面向移动和嵌入式视觉应用的高效卷积神经网络

📄 arXiv 2017
👥 Andrew G. Howard et al.
🏢 Google Inc.
📅 April 2017
🔖 28,000+ Citations
📄 arXiv 💻 TensorFlow 🔥 PyTorch Hub

问题背景

📱 移动设备的限制

传统深度学习模型需要大量计算资源和存储空间，无法在移动设备、嵌入式系统和物联网设备上实时运行。

⚡ 延迟与功耗

在AR/VR、自动驾驶、机器人等应用中，识别任务需要在计算受限的平台上及时完成，功耗是关键约束。

📊 精度vs效率

追求更高准确率往往意味着更深更复杂的网络，但这与移动端部署的需求相矛盾。

核心洞察：移动和嵌入式视觉应用需要小型、低延迟的模型，能够在资源受限的环境中高效运行，同时保持合理的准确率。

核心创新

MobileNet 引入了革命性的深度可分离卷积（Depthwise Separable Convolution），将标准卷积分解为两个更高效的运算：

标准卷积计算成本
D_K × D_K × M × N × D_F × D_F
↓ 分解为
深度可分离卷积计算成本
D_K × D_K × M × D_F × D_F + M × N × D_F × D_F
计算量减少约 8-9倍（3×3卷积核时）

🔍 Depthwise 卷积

每个输入通道应用一个单独的卷积核，独立提取空间特征，不混合通道信息。

🔗 Pointwise 卷积

使用1×1卷积跨通道组合特征，将深度卷积的输出投影到新的通道空间。

📐 Width Multiplier (α)

全局超参数，按比例减少每层的通道数。α ∈ {1.0, 0.75, 0.5, 0.25}，实现精度-延迟的灵活权衡。

📏 Resolution Multiplier (ρ)

减少输入图像分辨率，进一步降低计算量。常用值：224, 192, 160, 128。

实验结果

MobileNet 在 ImageNet 分类任务上实现了与大型网络相当的准确率，但计算成本大幅降低：

模型 参数量 MAdds Top-1 准确率 Top-5 准确率
VGG-16 138M 15,500M 71.5% 90.0%
AlexNet 60M 720M 57.2% 80.2%
1.0 MobileNet-224 4.2M 569M 70.6% 89.5%
0.5 MobileNet-160 1.3M 150M 59.1% 81.9%
0.25 MobileNet-128 0.5M 41M 50.6% 75.0%

效率突破：1.0 MobileNet-224 仅用 4.2M 参数（VGG-16 的 3%）和 569M MAdds（VGG-16 的 3.7%），达到了接近的准确率（70.6% vs 71.5%）。

目标检测与细粒度分类

MobileNet 在 COCO 目标检测任务中，作为 SSD 和 Faster R-CNN 的骨干网络，仅消耗一小部分计算量就达到与其他网络相当的结果。

人脸嵌入

通过知识蒸馏从 FaceNet（1.6B MAdds）训练 MobileNet，实现 286M MAdds 的模型达到 79.4% 准确率，大幅压缩模型体积。

影响与意义

  • 移动AI的开创者

    首次成功将深度学习带入移动设备，使实时图像识别、AR/VR应用在智能手机上成为可能。

  • 边缘计算的基石

    为物联网设备、嵌入式系统和边缘AI提供了高效的神经网络架构范式。

  • 可缩放架构的先驱

    引入 width multiplier 和 resolution multiplier 概念，开创了"一个架构，多种规模"的设计模式。

  • 产业应用广泛

    成为移动端视觉应用的标准 backbone，应用于 Google Photos、设备端翻译、实时物体检测等场景。

  • 后续演进

    催生了 MobileNetV2（Inverted Residuals）、MobileNetV3（NAS搜索）、MobileNetV4等改进版本，持续引领移动端模型设计。

代码实现

🐍 PyTorch MobileNet V1 实现 
import torch
import torch.nn as nn

class DepthwiseSeparableConv(nn.Module):
    """深度可分离卷积 = Depthwise卷积 + Pointwise卷积"""
    def __init__(self, in_channels, out_channels, stride=1):
        super().__init__()
        # Depthwise卷积：每个通道独立卷积
        self.depthwise = nn.Conv2d(
            in_channels, in_channels, 3,
            stride=stride, padding=1, groups=in_channels,
            bias=False
        )
        self.bn1 = nn.BatchNorm2d(in_channels)
        
        # Pointwise卷积：1×1卷积跨通道组合
        self.pointwise = nn.Conv2d(
            in_channels, out_channels, 1,
            stride=1, bias=False
        )
        self.bn2 = nn.BatchNorm2d(out_channels)
        self.relu = nn.ReLU6(inplace=True)  # MobileNet使用ReLU6
    
    def forward(self, x):
        x = self.relu(self.bn1(self.depthwise(x)))
        x = self.relu(self.bn2(self.pointwise(x)))
        return x


class MobileNetV1(nn.Module):
    """MobileNet V1 完整实现"""
    def __init__(self, num_classes=1000, width_mult=1.0):
        super().__init__()
        
        # 配置：输入通道, 输出通道, stride
        # 使用width multiplier缩放通道数
        def make_divisible(v, divisor=8):
            return int((v + divisor // 2) // divisor * divisor)
        
        cfg = [
            (32, 64, 1),    # conv1之后的第一个块
            (64, 128, 2),   # 下采样
            (128, 128, 1),
            (128, 256, 2),  # 下采样
            (256, 256, 1),
            (256, 512, 2),  # 下采样
            (512, 512, 1),  # 重复5次
            (512, 512, 1),
            (512, 512, 1),
            (512, 512, 1),
            (512, 512, 1),
            (512, 1024, 2), # 下采样
            (1024, 1024, 1),
        ]
        
        # 初始卷积层
        input_channel = make_divisible(32 * width_mult)
        self.conv1 = nn.Sequential(
            nn.Conv2d(3, input_channel, 3, stride=2, padding=1, bias=False),
            nn.BatchNorm2d(input_channel),
            nn.ReLU6(inplace=True)
        )
        
        # 构建深度可分离卷积层
        layers = []
        for in_c, out_c, stride in cfg:
            in_c = make_divisible(in_c * width_mult)
            out_c = make_divisible(out_c * width_mult)
            layers.append(DepthwiseSeparableConv(in_c, out_c, stride))
        
        self.features = nn.Sequential(*layers)
        
        # 分类头
        self.avgpool = nn.AdaptiveAvgPool2d(1)
        self.classifier = nn.Sequential(
            nn.Dropout(0.2),
            nn.Linear(make_divisible(1024 * width_mult), num_classes)
        )
        
        self._initialize_weights()
    
    def _initialize_weights(self):
        for m in self.modules():
            if isinstance(m, nn.Conv2d):
                nn.init.kaiming_normal_(m.weight, mode='fan_out')
                if m.bias is not None:
                    nn.init.zeros_(m.bias)
            elif isinstance(m, nn.BatchNorm2d):
                nn.init.ones_(m.weight)
                nn.init.zeros_(m.bias)
            elif isinstance(m, nn.Linear):
                nn.init.normal_(m.weight, 0, 0.01)
                nn.init.zeros_(m.bias)
    
    def forward(self, x):
        x = self.conv1(x)
        x = self.features(x)
        x = self.avgpool(x)
        x = torch.flatten(x, 1)
        x = self.classifier(x)
        return x


def create_mobilenet(alpha=1.0, num_classes=1000):
    """
    创建不同规模的 MobileNet
    alpha (width multiplier): 1.0, 0.75, 0.5, 0.25
    """
    return MobileNetV1(num_classes=num_classes, width_mult=alpha)

from torchvision import models

# 标准 MobileNet V1
model = models.mobilenet_v2(pretrained=True)  # torchvision提供的是v2

# 推理示例
model.eval()
with torch.no_grad():
    dummy_input = torch.randn(1, 3, 224, 224)
    output = model(dummy_input)
    predictions = torch.softmax(output, dim=1)
    top5 = torch.topk(predictions, 5)
    print(f"Top-5 预测: {top5.indices[0].tolist()}")
