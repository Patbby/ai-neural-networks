📄 Paper Information

Full Title: Going Deeper with Convolutions

Authors:

Christian Szegedy Wei Liu Yangqing Jia Pierre Sermanet Scott Reed Dragomir Anguelov Dumitru Erhan Vincent Vanhoucke Andrew Rabinovich

Abstract: We propose a deep convolutional neural network architecture codenamed "Inception", which was responsible for setting the new state of the art for classification and detection in the ImageNet Large-Scale Visual Recognition Challenge 2014 (ILSVRC 2014). The main hallmark of this architecture is the improved utilization of the computing resources inside the network. This was achieved by a carefully crafted design that allows for increasing the depth and width of the network while keeping the computational budget constant.

Key Achievement: Winner of ILSVRC 2014 with a top-5 error rate of 6.67%, significantly outperforming previous approaches while using only 4 million parameters (compared to AlexNet's 60 million).

🏗️ Architecture Overview

🔀 Inception Module

The core innovation - parallel convolutional filters of different sizes (1×1, 3×3, 5×5) plus pooling, all computed simultaneously. This allows the network to capture features at multiple scales, from fine details to broader patterns, in a single layer.

📉 1×1 Dimension Reduction

Before expensive 3×3 and 5×5 convolutions, 1×1 convolutions reduce the number of input channels. This "bottleneck" design dramatically cuts computational cost while preserving representational power. A key technique later adopted by ResNet.

🎯 Global Average Pooling

Replacing fully connected layers with global average pooling reduced parameters from millions to thousands. Each feature map is averaged to produce a single value, naturally producing 1000 outputs for ImageNet classes. More interpretable and less prone to overfitting.

⚡ Auxiliary Classifiers

Two additional softmax classifiers at intermediate layers (Inception 4a and 4d) combat the vanishing gradient problem in deep networks. These provide additional gradient signal during training and are discarded during inference. Improved top-1 accuracy by 0.4%.

💡 Hebbian Principle

The architecture design follows the neuroscience-inspired Hebbian principle: "neurons that fire together, wire together." By clustering correlated outputs and concentrating on the most promising regions, the network achieves both depth and efficiency.

📊 Efficiency Breakthrough

Achieved AlexNet-level accuracy with only 4 million parameters (vs. 60 million). The careful balance of depth (22 layers) and computational efficiency made deep networks practical for deployment on resource-constrained devices.

📊 Impact & Legacy

Immediate Impact: GoogLeNet won ILSVRC 2014, establishing a new paradigm for efficient deep learning. The 6.67% top-5 error rate was a significant improvement over the previous state-of-the-art.

Key Contributions to the Field:

  • Multi-scale Processing: The Inception module's parallel filter approach became a template for future architectures
  • 1×1 Convolutions: Popularized the use of 1×1 convolutions for dimension reduction, later essential in ResNet and MobileNet
  • Global Average Pooling: Demonstrated that FC layers could be eliminated, reducing overfitting and parameters
  • Efficiency Focus: Proved that careful architectural design beats simply adding more parameters

Successors: The Inception family evolved through v2 (with batch normalization), v3 (factorized convolutions), v4 (combined with ResNet ideas), and Xception (depthwise separable convolutions).

Modern Relevance: Concepts from GoogLeNet remain fundamental: 1×1 bottlenecks appear in ResNet, EfficientNet, and Vision Transformers. The multi-scale philosophy influences modern architectures like HRNet.
