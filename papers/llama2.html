📋 Abstract

Llama 2 is a collection of pretrained and fine-tuned large language models from Meta AI, ranging from 7B to 70B parameters. The fine-tuned models, Llama 2-Chat, are optimized for dialogue use cases and outperform open-source chat models on most benchmarks while being comparable to some closed-source models like ChatGPT.

"We are releasing Llama 2 to the public for research and commercial use — the first fully open LLM to challenge GPT-3.5 on head-to-head human preference evaluations."

⚙️ Architecture Improvements over Llama 1

  • Grouped Query Attention (GQA): 70B model uses GQA for faster inference — reduces KV cache memory
  • Longer context: 4096 tokens (2× Llama 1's 2048)
  • RoPE positional embeddings: Rotary Position Embedding for better length generalization
  • Pre-normalization (RMSNorm): Normalize input rather than output for training stability
  • SwiGLU activation: Gated linear unit variant for improved performance

🛡️ Safety Focus

A key differentiator of Llama 2-Chat is its extensive safety work:

  • Dual reward models: Separate models for helpfulness and safety — enable trade-off control
  • Ghost Attention (GAtt): System prompt conditioning persists across long conversations
  • Red teaming: 3,000+ attack prompts evaluated across model versions
  • Human evaluation: 4,000 annotations comparing safety vs GPT-3.5 and ChatGPT

💡 Impact & Legacy

  • First open-source LLM with serious commercial license — sparked ecosystem of fine-tuned variants
  • Enabled Mistral, WizardLM, CodeLlama, Orca, and hundreds of community fine-tunes
  • Established blueprint for open RLHF pipelines (SFT → RM → PPO)
  • Led to Llama 3 (2024) with 8B/70B/405B sizes and GPT-4 competitive performance
