Bahdanau Attention: The foundation of modern attention mechanisms that revolutionized sequence modeling

📅 ICLR 2015 👤 Dzmitry Bahdanau, Kyunghyun Cho, Yoshua Bengio 🏫 Université de Montréal
📄 arXiv 🏛️ ICLR 2015 📊 35,000+ Citations

The Bottleneck Problem

🎯 Information Compression Bottleneck

Traditional encoder-decoder (Seq2Seq) models compress the entire source sentence into a single fixed-length vector. As sentence length increases, this becomes a severe bottleneck—information from earlier words gets diluted or lost entirely. The model struggles with long sentences, showing dramatic performance degradation as length increases.

The Attention Solution

💡 Dynamic Context Vector

Instead of a single fixed context vector, Bahdanau Attention computes a different context vector for each decoder step. The decoder "looks at" different parts of the source sentence when generating each target word, using a learned alignment (attention weights) to determine where to focus.

Context Vector: cᵢ = Σⱼ αᵢⱼ · hⱼ
Attention Weights: αᵢⱼ = softmax(eᵢⱼ) = exp(eᵢⱼ) / Σₖ exp(eᵢₖ)
Alignment Score: eᵢⱼ = a(sᵢ₋₁, hⱼ) — learned by a feedforward network

Attention Heatmap Visualization

The paper demonstrated that the learned attention weights create linguistically plausible alignments between source and target words. Even with different word orders between languages, the model learns correct soft alignments.

PyTorch Implementation

# Bahdanau (Additive) Attention Implementation
import torch
import torch.nn as nn
import torch.nn.functional as F

class BahdanauAttention(nn.Module):
    def __init__(self, hidden_size):
        super().__init__()
        self.W_s = nn.Linear(hidden_size, hidden_size)  # Decoder state
        self.W_h = nn.Linear(hidden_size, hidden_size)  # Encoder hidden
        self.V = nn.Linear(hidden_size, 1, bias=False)  # Score
    
    def forward(self, decoder_hidden, encoder_outputs):
        # decoder_hidden: (batch, hidden_size)
        # encoder_outputs: (batch, seq_len, hidden_size)
        
        # Expand decoder hidden to match encoder sequence length
        decoder_hidden = decoder_hidden.unsqueeze(1)  # (batch, 1, hidden)
        
        # Calculate alignment scores e_ij = V * tanh(W_s * s + W_h * h)
        score = torch.tanh(
            self.W_s(decoder_hidden) + self.W_h(encoder_outputs)
        )  # (batch, seq_len, hidden)
        
        scores = self.V(score).squeeze(-1)  # (batch, seq_len)
        
        # Softmax to get attention weights
        attn_weights = F.softmax(scores, dim=1)  # (batch, seq_len)
        
        # Compute context vector as weighted sum
        context = torch.bmm(
            attn_weights.unsqueeze(1),  # (batch, 1, seq_len)
            encoder_outputs                     # (batch, seq_len, hidden)
        ).squeeze(1)  # (batch, hidden)
        
        return context, attn_weights
