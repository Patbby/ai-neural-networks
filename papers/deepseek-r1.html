📋 Abstract

DeepSeek-R1 introduces a novel approach to incentivize reasoning in large language models through reinforcement learning, achieving performance comparable to OpenAI o1. The key insight is that reasoning capabilities can emerge from RL training without supervised fine-tuning on chain-of-thought data.

"We first train DeepSeek-R1-Zero — a model that acquires reasoning capabilities through pure RL without any supervised fine-tuning. It naturally develops behaviors like self-verification, reflection, and aha moments."

⚙️ Group Relative Policy Optimization (GRPO)

DeepSeek-R1 uses GRPO instead of PPO to reduce memory overhead:

  • Group sampling: Sample G outputs per question, use group mean as baseline
  • No critic model: Eliminates the value network — halves memory usage
  • Rule-based reward: Accuracy (math/code verifiable) + Format (think tags) — no reward model needed
  • KL penalty: Prevents over-optimization away from reference policy

💬 Reasoning Chain Format

R1 uses special tags to separate chain-of-thought reasoning from the final answer:

<think>
Let me solve this step by step...
The problem asks for the sum of integers from 1 to 100.
Using the formula n(n+1)/2 where n=100:
= 100 × 101 / 2
= 10100 / 2
= 5050
Let me verify: (1+100)×50 = 101×50 = 5050 ✓
</think>

<answer>
The sum of integers from 1 to 100 is 5050.
</answer>

The model is rewarded for correct final answers inside <answer> tags, not for the reasoning process itself — leading to emergent self-correction and reflection behaviors.

💡 Key Contributions

  • R1-Zero: First model to develop chain-of-thought reasoning via pure RL, no SFT needed — "aha moment" phenomenon observed
  • GRPO: Critic-free policy optimization that scales to 671B parameter models
  • Distillation: Knowledge distilled into smaller models (1.5B–70B) that outperform much larger counterparts
  • Open weights: Full model weights released — enabling the open-source community to build on top
  • Cost efficiency: Training costs ~$5.5M — 30x cheaper than comparable closed models
