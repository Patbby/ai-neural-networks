历史背景

1989
LeCun 等人设计了首个用于手写邮政编码识别的卷积神经网络，使用反向传播算法训练，应用于美国邮政服务。
1993
LeNet-1 演示：首个能够以良好速度和准确率识别手写数字的卷积网络。
1998
发表经典论文《Gradient-Based Learning Applied to Document Recognition》，LeNet-5 成为深度学习"Hello World"。
2000s-2010s
LeNet 的成功启发了 AlexNet、ResNet 等后续架构，奠定了现代计算机视觉的基础。

问题陈述

🔍 传统方法的局限

在 LeNet 出现之前，手写字符识别主要依赖手工设计的特征提取器（如模板匹配、边缘检测），这些方法无法自动学习数据的内在表示，泛化能力差。

💡 深度学习的曙光

LeCun 等人提出了一种端到端的梯度学习方法，让神经网络自动从原始像素学习层次化的特征表示，开创了现代深度学习范式。

核心洞察：卷积操作配合可学习的滤波器，能够以较少的参数有效提取图像在多个位置的局部特征。在当时计算资源极其有限的条件下，效率是至关重要的。

核心创新

  • 🎯

    卷积神经网络架构

    首次系统化展示了 CNN 的核心组件：卷积层提取局部特征，池化层降低维度，全连接层完成分类。

  • 🔗

    反向传播训练

    成功将反向传播算法应用于卷积层，证明了深层网络可以通过梯度下降进行端到端训练。

  • ⚡

    参数共享与稀疏连接

    卷积核在整张图像上共享权重，大幅减少了参数量，使深层网络在当时硬件条件下成为可能。

  • 🏭

    商业成功应用

    被多家银行用于支票手写数字识别，处理了美国约10%-20%的支票，证明了深度学习在实际工业场景中的价值。

网络架构

LeNet-5 包含 7 层（不含输入）：2 个卷积层、2 个池化层、3 个全连接层，约 60,000 个参数。

INPUT (1×32×32) 灰度图像
↓
C1: Conv (6@28×28) 5×5 kernel
↓
S2: Avg Pool (6@14×14) 2×2 pool
↓
C3: Conv (16@10×10) 5×5 kernel
↓
S4: Avg Pool (16@5×5) 2×2 pool
↓
C5: Conv (120@1×1) 5×5 kernel
↓
F6: FC (84) 全连接
↓
OUTPUT (10) RBF / Softmax

设计细节：原始 LeNet-5 使用缩放双曲正切激活函数（scaled tanh）防止梯度消失，输出层采用欧几里得径向基函数（RBF）单元。C3 层采用稀疏连接策略，并非所有特征图都与 S2 层连接，这是为了控制参数量与训练数据规模相匹配。

PyTorch 实现

# LeNet-5 PyTorch 实现 (现代版本，使用 ReLU 和 MaxPool)
import torch
import torch.nn as nn
import torch.nn.functional as F

class LeNet5(nn.Module):
    def __init__(self, num_classes=10):
        super(LeNet5, self).__init__()
        # 卷积层部分
        self.conv1 = nn.Conv2d(1, 6, kernel_size=5, padding=2)  # 1x32x32 -> 6x28x28
        self.conv2 = nn.Conv2d(6, 16, kernel_size=5)  # 6x14x14 -> 16x10x10
        self.conv3 = nn.Conv2d(16, 120, kernel_size=5)  # 16x5x5 -> 120x1x1
        
        # 全连接层部分
        self.fc1 = nn.Linear(120, 84)
        self.fc2 = nn.Linear(84, num_classes)
    
    def forward(self, x):
        # 卷积 + 激活 + 池化
        x = F.max_pool2d(F.relu(self.conv1(x)), 2)  # 6x28x28 -> 6x14x14
        x = F.max_pool2d(F.relu(self.conv2(x)), 2)  # 16x10x10 -> 16x5x5
        x = F.relu(self.conv3(x))  # 120x1x1
        
        # 展平 + 全连接
        x = x.view(x.size(0), -1)
        x = F.relu(self.fc1(x))
        x = self.fc2(x)
        return x

# 模型实例化
model = LeNet5(num_classes=10)
print(f"模型参数量: {sum(p.numel() for p in model.parameters())/1000:.1f}K")
