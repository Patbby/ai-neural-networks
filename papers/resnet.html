🎯 核心问题：退化问题 (Degradation Problem)

退化问题：

随着神经网络深度增加，训练准确率反而饱和甚至下降。这不是过拟合——训练误差和测试误差都变差了。

56层网络的训练误差和测试误差都高于20层网络，尽管56层网络有更大的容量。

退化问题可视化：普通网络 vs ResNet

普通网络越深性能越差，ResNet越深性能越好

传统观点认为深层网络难以优化是因为梯度消失/爆炸。但Batch Normalization很大程度上解决了这个问题。ResNet揭示了一个更深层的问题：

核心洞察： 让网络直接学习恒等映射(identity mapping)是非常困难的。如果多层非线性层渐进地逼近恒等映射，其残差(residual)会趋近于零，这比直接学习恒等映射更容易。

💡 解决方案：残差学习框架

ResNet的创新： 不再让网络学习期望的底层映射 H(x)，而是学习残差 F(x) = H(x) - x。原始映射变为 F(x) + x。
y = F(x, {Wᵢ}) + x

其中 F(x, {Wᵢ}) 表示要学习的残差映射。x 是跳跃连接(skip connection)提供的恒等映射。

🏗️ 残差块架构详解

残差块结构对比：普通块 vs 残差块

两种残差块设计

Basic Block（用于ResNet-18/34）： 两个3×3卷积层，适用于较浅网络。

Bottleneck Block（用于ResNet-50/101/152）： 1×1 → 3×3 → 1×1 三层结构，先降维再卷积最后升维，减少计算量。

Bottleneck 结构 (ResNet-50+)

通过1×1卷积先降维再升维，将3×3卷积的计算量减少约50%

为什么跳跃连接有效？梯度高速公路

跳跃连接创建"梯度高速公路"，让深层网络也能有效训练

💻 PyTorch 完整实现

import torch import torch.nn as nn class BasicBlock(nn.Module): """ResNet-18/34 使用的残差块 - 两个3×3卷积""" expansion = 1 def __init__(self, in_channels, out_channels, stride=1): super(BasicBlock, self).__init__() # 第一个3×3卷积，可能下采样 self.conv1 = nn.Conv2d(in_channels, out_channels, kernel_size=3, stride=stride, padding=1, bias=False) self.bn1 = nn.BatchNorm2d(out_channels) # 第二个3×3卷积 self.conv2 = nn.Conv2d(out_channels, out_channels, kernel_size=3, stride=1, padding=1, bias=False) self.bn2 = nn.BatchNorm2d(out_channels) self.relu = nn.ReLU(inplace=True) # 下采样层：当维度不匹配时使用1×1卷积 self.shortcut = nn.Sequential() if stride != 1 or in_channels != out_channels: self.shortcut = nn.Sequential( nn.Conv2d(in_channels, out_channels, kernel_size=1, stride=stride, bias=False), nn.BatchNorm2d(out_channels) ) def forward(self, x): identity = x out = self.conv1(x) out = self.bn1(out) out = self.relu(out) out = self.conv2(out) out = self.bn2(out) # 关键：跳跃连接 + 残差学习 F(x) + x out += self.shortcut(identity) out = self.relu(out) return out class Bottleneck(nn.Module): """ResNet-50/101/152 使用的瓶颈块 - 1×1→3×3→1×1""" expansion = 4 def __init__(self, in_channels, out_channels, stride=1): super(Bottleneck, self).__init__() # 1×1 降维 self.conv1 = nn.Conv2d(in_channels, out_channels, kernel_size=1, bias=False) self.bn1 = nn.BatchNorm2d(out_channels) # 3×3 卷积 self.conv2 = nn.Conv2d(out_channels, out_channels, kernel_size=3, stride=stride, padding=1, bias=False) self.bn2 = nn.BatchNorm2d(out_channels) # 1×1 升维 self.conv3 = nn.Conv2d(out_channels, out_channels * 4, kernel_size=1, bias=False) self.bn3 = nn.BatchNorm2d(out_channels * 4) self.relu = nn.ReLU(inplace=True) # 下采样层 self.shortcut = nn.Sequential() if stride != 1 or in_channels != out_channels * 4: self.shortcut = nn.Sequential( nn.Conv2d(in_channels, out_channels * 4, kernel_size=1, stride=stride, bias=False), nn.BatchNorm2d(out_channels * 4) ) def forward(self, x): identity = x out = self.conv1(x) out = self.bn1(out) out = self.relu(out) out = self.conv2(out) out = self.bn2(out) out = self.relu(out) out = self.conv3(out) out = self.bn3(out) out += self.shortcut(identity) out = self.relu(out) return out # 使用预训练模型进行迁移学习 from torchvision import models # 加载预训练ResNet-50 resnet50 = models.resnet50(pretrained=True) # 冻结特征提取层 for param in resnet50.parameters(): param.requires_grad = False # 替换最后的全连接层进行微调 resnet50.fc = nn.Linear(resnet50.fc.in_features, num_classes)

🌍 影响与后续发展

2015
ResNet 发表
提出残差学习，152层网络赢得ILSVRC冠军
2016
ResNet v2
Identity Mappings改进，将BN和ReLU前置到卷积层前

Wide ResNet
增加通道宽度而非深度，性能更优
2017
ResNeXt
引入Cardinality维度，使用分组卷积
2017
DenseNet
将跳跃连接发挥到极致，每层连接所有前面层
2019
EfficientNet
复合缩放深度、宽度、分辨率

ResNet的影响远远超出了图像分类：

  • 目标检测： ResNet成为Faster R-CNN、Mask R-CNN等检测框架的默认backbone
  • 语义分割： FCN、U-Net、DeepLab等使用ResNet作为编码器
  • 视觉Transformer： ViT的架构设计深受ResNet影响
  • NLP领域： Transformer的残差连接直接继承自ResNet
  • 生成模型： Stable Diffusion的U-Net使用ResNet-style blocks

📚 关键启示

ResNet给我们的四个启示：

1. 问题重构： 学习残差比学习原始映射更容易

2. 梯度高速公路： 跳跃连接为梯度提供直接回传路径

3. 深度即力量： 解决了退化问题后，更深确实更好

4. 简洁即优雅： 一个简单idea解决了困扰学界多年的问题
