🎯 Problem Statement

Before ResNet, deep neural networks faced a critical limitation known as the degradation problem: as network depth increased, accuracy saturated and then degraded rapidly. This was surprising because deeper networks should have at least as much representational power as shallower ones.

The degradation problem was not caused by overfitting — training error actually increased with depth. This indicated that deeper networks were harder to optimize, not that they lacked capacity to learn.

Degradation: Training Error↑ as Depth↑
Adding more layers led to higher training error — the opposite of what was expected

Key Questions

❓ Is learning better networks as easy as stacking more layers?
❓ Why do deeper networks become harder to train?
❓ How can we enable training of networks with hundreds or thousands of layers?

💡 Key Innovation

ResNet introduced the concept of residual learning: instead of learning the desired mapping H(x) directly, the layers learn a residual function F(x) = H(x) - x. The original mapping is then reconstructed as:

H(x) = F(x) + x
Where F(x) is the residual mapping to be learned, and x is the skip connection (identity mapping)

The key insight is that learning residual functions is easier than learning the original unreferenced functions. If the optimal mapping is close to an identity mapping, it's easier to push the residual to zero than to fit an identity mapping by a stack of nonlinear layers.

Why This Works

1. Gradient Highway: Skip connections create direct paths for gradient flow during backpropagation, mitigating vanishing gradients.

2. Identity Fallback: If residual layers learn nothing, the network can default to identity mapping (F(x) = 0), ensuring deeper networks perform at least as well as shallower ones.

3. Decomposed Learning: The network learns "what needs to change" rather than "the complete transformation" — a simpler optimization problem.

💻 PyTorch Implementation

Python resnet_block.py 
import torch
import torch.nn as nn

class BasicBlock(nn.Module):
    """Basic Residual Block for ResNet-18/34"""
    expansion = 1
    
    def __init__(self, in_channels, out_channels, stride=1):
        super(BasicBlock, self).__init__()
        self.conv1 = nn.Conv2d(in_channels, out_channels, kernel_size=3, 
                               stride=stride, padding=1, bias=False)
        self.bn1 = nn.BatchNorm2d(out_channels)
        self.relu = nn.ReLU(inplace=True)
        self.conv2 = nn.Conv2d(out_channels, out_channels, kernel_size=3,
                               stride=1, padding=1, bias=False)
        self.bn2 = nn.BatchNorm2d(out_channels)
        
        # Shortcut connection
        self.shortcut = nn.Sequential()
        if stride != 1 or in_channels != out_channels:
            self.shortcut = nn.Sequential(
                nn.Conv2d(in_channels, out_channels, kernel_size=1,
                         stride=stride, bias=False),
                nn.BatchNorm2d(out_channels)
            )
    
    def forward(self, x):
        identity = x
        
        out = self.conv1(x)
        out = self.bn1(out)
        out = self.relu(out)
        
        out = self.conv2(out)
        out = self.bn2(out)
        
        # Key: F(x) + x
        out += self.shortcut(identity)
        out = self.relu(out)
        
        return out


class Bottleneck(nn.Module):
    """Bottleneck Block for ResNet-50/101/152"""
    expansion = 4
    
    def __init__(self, in_channels, out_channels, stride=1):
        super(Bottleneck, self).__init__()
        self.conv1 = nn.Conv2d(in_channels, out_channels, kernel_size=1, bias=False)
        self.bn1 = nn.BatchNorm2d(out_channels)
        self.conv2 = nn.Conv2d(out_channels, out_channels, kernel_size=3,
                               stride=stride, padding=1, bias=False)
        self.bn2 = nn.BatchNorm2d(out_channels)
        self.conv3 = nn.Conv2d(out_channels, out_channels * 4, kernel_size=1, bias=False)
        self.bn3 = nn.BatchNorm2d(out_channels * 4)
        self.relu = nn.ReLU(inplace=True)
        
        self.shortcut = nn.Sequential()
        if stride != 1 or in_channels != out_channels * 4:
            self.shortcut = nn.Sequential(
                nn.Conv2d(in_channels, out_channels * 4, kernel_size=1,
                         stride=stride, bias=False),
                nn.BatchNorm2d(out_channels * 4)
            )
    
    def forward(self, x):
        identity = x
        
        out = self.relu(self.bn1(self.conv1(x)))
        out = self.relu(self.bn2(self.conv2(out)))
        out = self.bn3(self.conv3(out))
        
        out += self.shortcut(identity)
        out = self.relu(out)
        
        return out

Key Training Details

• Batch Normalization: Applied after every convolution, before activation

• Weight Decay: 0.0001

• Momentum: 0.9

• Learning Rate: Start at 0.1, divided by 10 when error plateaus

• Data Augmentation: Random crop, horizontal flip, color jittering
