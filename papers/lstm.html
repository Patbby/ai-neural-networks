LSTM

Long Short-Term Memory — The architecture that solved the vanishing gradient problem and made long-range sequence learning possible.

📄 Neural Computation 1997 👥 Hochreiter & Schmidhuber 🏷️ Recurrent Neural Networks

The Problem: Vanishing Gradients

Before LSTM, Recurrent Neural Networks (RNNs) were theoretically capable of learning from sequential data, but they faced a critical limitation: the vanishing gradient problem. When training on long sequences, the gradients that flow backward through time would either shrink exponentially (vanish) or grow uncontrollably (explode), making it impossible to learn relationships between events separated by many time steps.

The Mathematical Barrier

In standard RNNs, the hidden state update is:

ht = tanh(Whht-1 + Wxxt + b)

During backpropagation through time, gradients must flow through this nonlinear activation repeatedly. After just 10-20 time steps, gradients become vanishingly small, preventing the network from learning long-range dependencies.

The LSTM Solution

LSTM solved the vanishing gradient problem through a brilliant architectural innovation: the memory cell. Instead of rewriting the entire hidden state at each step, LSTM maintains a separate cell state that acts as a "conveyor belt" for information, with minimal modifications as it flows through time.

🔴 Forget Gate (ft)

Decides what information to discard from the cell state. Uses a sigmoid function to output values between 0 (forget) and 1 (keep).

ft = σ(Wf · [ht-1, xt] + bf)

🟠 Input Gate (it)

Decides what new information to store in the cell state. Works with the candidate values to update the memory.

it = σ(Wi · [ht-1, xt] + bi)

🟣 Candidate Values (C̃t)

New candidate values that could be added to the cell state. Created using a tanh activation.

C̃t = tanh(WC · [ht-1, xt] + bC)

🔵 Output Gate (ot)

Decides what parts of the cell state to output as the hidden state. Filters the cell state through a sigmoid.

ot = σ(Wo · [ht-1, xt] + bo)

Cell State Update

The cell state is updated by combining the forgotten past with new candidate information:

ct = ft ⊙ ct-1 + it ⊙ C̃t

The hidden state (output) is computed from the filtered cell state:

ht = ot ⊙ tanh(ct)

Key Innovations

1. Constant Error Carousel (CEC)

The core innovation of LSTM is the Constant Error Carousel — the cell state's ability to maintain information across many time steps with minimal modification. The self-connected recurrent edge with weight 1.0 ensures that gradients can flow backward through time without vanishing or exploding.

2. Gating Mechanisms

Unlike standard RNNs that completely rewrite their hidden state, LSTMs use learned gates to selectively read, write, and erase information. This "learned memory management" allows the network to decide which information is worth keeping and which should be discarded.

3. Separation of Memory and Output

LSTM explicitly separates the cell state (long-term memory) from the hidden state (output/short-term memory). This separation allows the network to maintain information internally without necessarily exposing it at every step.

Why LSTM Still Matters

  • Inductive bias for sequences: LSTMs naturally model temporal dependencies without massive datasets
  • Resource efficiency: Smaller models suitable for edge devices and mobile
  • Streaming applications: Natural left-to-right processing for real-time systems
  • Interpretability: Gate activations provide insights into what the model remembers/forgets
  • Foundation for modern architectures: Gating concepts influenced Transformers, Mamba, and more

Variants & Extensions

Peephole LSTM (2000)

Gates can "peep" at the cell state directly, giving them more information for their decisions.

Coupled Gates

Forget and input gates are coupled: the network only inputs new values when it forgets something.

Bidirectional LSTM

Two LSTMs run in parallel — one forward, one backward. Combines outputs for full context.

Stacked/Deep LSTM

Multiple LSTM layers stacked vertically to learn hierarchical representations.
