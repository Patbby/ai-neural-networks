Friedman's seminal framework connecting boosting to gradient descent in function space. The theoretical foundation powering XGBoost, LightGBM, and modern gradient boosting machines.

Machine Learning 2001

Random Forests

Leo Breiman's ensemble masterpiece combining bagging with random feature selection. Creates uncorrelated decision trees that vote together, achieving unprecedented accuracy with built-in OOB validation.

Gradient Boosting Machine

Jerome Friedman's optimization framework that views boosting as gradient descent in function space. Fits weak learners to pseudo-residuals, enabling any differentiable loss function and spawning XGBoost, LightGBM, and CatBoost.

ICML 1997

AdaBoost

Freund & Schapire's adaptive boosting algorithm that sequentially trains weak learners on reweighted data, focusing on misclassified examples. The first practical boosting method that sparked the ensemble learning revolution.

ICLR 2019

SENet

ICLR 2017

NAS

ICLR 2015

Bahdanau Attention

Neural Machine Translation by Jointly Learning to Align and Translate. The foundational attention mechanism that gave Seq2Seq models "new life" — introduced dynamic context vectors and soft alignment learning.

NAACL 2018 (Best Paper)

ELMo

Deep Contextualized Word Representations - Bidirectional LSTM-based context-dependent embeddings. First to show that pretrained contextualized representations dramatically improve NLP task performance.

A ConvNet for the 2020s - The pure ConvNet that rivaled Vision Transformers. Starting from ResNet-50, modernizing step-by-step with Transformer-inspired design choices to achieve 87.8% ImageNet accuracy without self-attention.

A simple, scalable, and stable policy gradient method for reinforcement learning - Uses clipped surrogate objective to prevent destructive large policy updates, becoming the go-to algorithm for RL.
