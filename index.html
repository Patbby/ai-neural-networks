📚 Classic Papers

PNAS 1982

Hopfield Network

Neural networks and physical systems with emergent collective computational abilities. Introduced associative memory through energy-based models, connecting neural networks to statistical mechanics. Foundation for modern energy-based models.

Nature 1986

Backpropagation

The foundational algorithm that made training deep neural networks possible. Introduced efficient gradient computation through reverse-mode automatic differentiation, enabling practical learning in multi-layer networks.

Nature 2016

AlphaGo

Mastering the game of Go with deep neural networks and tree search. First AI to defeat a world champion. Combined policy networks, value networks, and Monte Carlo Tree Search (MCTS).

NeurIPS 2022 (Oral)

InstructGPT

Training language models to follow instructions with human feedback. The RLHF breakthrough that powers ChatGPT - aligning AI with human intent through reinforcement learning.

NeurIPS 2023

QLoRA

Efficient Finetuning of Quantized LLMs. Revolutionary technique enabling 65B parameter model finetuning on a single 48GB GPU. Combines 4-bit NormalFloat quantization with Low-Rank Adaptation.

ICLR 2015

Adam Optimizer

A Method for Stochastic Optimization - The default optimizer for deep learning. Combines momentum with adaptive learning rates per parameter. Used in virtually every modern neural network.

NeurIPS 2019

RMSNorm

Root Mean Square Layer Normalization — A simpler, faster alternative to LayerNorm that removes mean centering. Used in modern LLMs like Llama, Chinchilla, and Gemma. 7%~64% speedup with comparable performance.

arXiv 2016

Layer Normalization

Simple normalization method to improve training speed for various neural network models. Computes mean and variance across features instead of batch. Essential for RNNs and Transformers.

ICML 2015

Batch Normalization

Accelerating Deep Network Training by Reducing Internal Covariate Shift. Enables higher learning rates, faster convergence, and training of deeper networks. Standard in all modern CNNs.

CVPR 2016

Inception v3

Rethinking the Inception Architecture - Factorized convolutions, asymmetric filters, label smoothing, and auxiliary classifiers. Achieved 3.58% top-5 error on ImageNet with remarkable efficiency.

NIPS 2014

Seq2Seq

Sequence to Sequence Learning with Neural Networks - The foundational encoder-decoder architecture that revolutionized machine translation and paved the way for modern NLP.

KDD 2016

XGBoost

eXtreme Gradient Boosting - The algorithm that dominated Kaggle competitions with second-order optimization, regularization, and systems innovations. The "Queen of Machine Learning."

Machine Learning 1995

Support-Vector Networks (SVM)

The foundational algorithm for maximum margin classification and kernel methods. Revolutionized machine learning with 50,000+ citations and the theoretical framework for generalization.

Proceedings of IEEE 1998

LeNet-5

Gradient-Based Learning Applied to Document Recognition - The CNN "Hello World" that started the deep learning revolution, used in banks for check recognition.

ICLR 2015

Adam Optimizer

A Method for Stochastic Optimization - The adaptive moment estimation algorithm that became the de facto standard optimizer for training deep neural networks with 100,000+ citations.

ICML 2015

Batch Normalization

Accelerating Deep Network Training by Reducing Internal Covariate Shift - The technique that revolutionized deep learning by enabling higher learning rates, faster convergence, and training of deeper networks.

ICML 2020

SimCLR

A Simple Framework for Contrastive Learning of Visual Representations - The breakthrough self-supervised learning method that showed contrastive learning can match supervised learning without memory banks or specialized architectures.

ECCV 2018

Group Normalization

A Simple Alternative to Batch Normalization for Small Batch Training - GN divides channels into groups and normalizes within each group, eliminating batch size constraints. 10.6% lower error than BN at batch size 2.

arXiv 2016

Instance Normalization

The Missing Ingredient for Fast Stylization - Per-instance normalization that enables real-time neural style transfer by normalizing each sample independently, preserving style while removing instance-specific contrast.

CVPR 2017

Feature Pyramid Networks (FPN)

Multi-scale object detection through hierarchical feature fusion. The top-down architecture with lateral connections that powers modern detectors with high-level semantics at all scales.

ICCV 2017 Best Paper

Mask R-CNN

A Painless, High-Quality, and Unified Framework for Object Detection and Instance Segmentation - Extends Faster R-CNN with a parallel mask prediction branch, achieving SOTA results with simple, clean architecture.

NeurIPS 2019

RMSNorm

Root Mean Square Layer Normalization - A simplified normalization that removes mean-centering from LayerNorm, offering 7-64% speedup while maintaining comparable performance. Widely adopted in LLaMA, Mistral, and modern LLMs.

JMLR 2014

Dropout

A Simple Way to Prevent Neural Networks from Overfitting - The breakthrough regularization technique that randomly deactivates neurons during training, forcing robust feature learning and preventing co-adaptation.

CVPR 2014

R-CNN

Rich Feature Hierarchies for Accurate Object Detection - The groundbreaking paper that combined CNNs with region proposals, revolutionizing object detection and establishing the transfer learning paradigm.

ICCV 2015

Fast R-CNN

Fast Region-based Convolutional Networks - Revolutionary object detection that shares conv computations across region proposals, achieving 213× speedup over R-CNN with higher accuracy.

arXiv 2015

Highway Networks

Training Very Deep Networks with Information Highways - The pioneering architecture with learned gating mechanisms that enabled 100+ layer networks and inspired ResNet's skip connections.

CVPR 2016 Best Paper

ResNet

Deep Residual Learning for Image Recognition - The breakthrough architecture enabling 100+ layer networks through skip connections.

ICCV 2023

DiT · Diffusion Transformers

Scalable Diffusion Models with Transformers - Replacing U-Nets with transformers for state-of-the-art image generation.

NAACL 2019

BERT

Bidirectional Encoder Representations from Transformers - Deep bidirectional pre-training that revolutionized NLP.

NeurIPS 2017

Transformer

Attention Is All You Need - The groundbreaking architecture that replaced RNNs with self-attention mechanisms.

NeurIPS 2022

Flash Attention

Fast and Memory-Efficient Exact Attention with IO-Awareness - Tiled attention algorithm that reduces HBM-SRAM transfers by 20x, enabling 64K sequence lengths without approximation and 3x speedup on GPT-2.

NeurIPS 2014

GAN

Generative Adversarial Networks - The revolutionary framework where two neural networks compete to generate realistic data.

ICML 2017

Wasserstein GAN

Wasserstein Generative Adversarial Networks - The breakthrough that replaced JS divergence with Earth Mover's Distance, solving vanishing gradients and providing meaningful loss metrics for stable GAN training.

NeurIPS 2017

WGAN-GP

Improved Training of Wasserstein GANs - Replaced problematic weight clipping with gradient penalty for stable GAN training. Enables 101-layer ResNets and language models with minimal hyperparameter tuning.

ICCV 2017

CycleGAN

Unpaired Image-to-Image Translation using Cycle-Consistent Adversarial Networks - Learn to translate without paired data.

CVPR 2017

Pix2Pix

Image-to-Image Translation with Conditional Adversarial Networks - The foundational paired image translation method using U-Net generator and PatchGAN discriminator with skip connections.

ICLR 2014

VAE

Auto-Encoding Variational Bayes - The probabilistic approach to deep generative modeling that learns structured latent representations.

NIPS 2012

AlexNet

ImageNet Classification with Deep Convolutional Neural Networks - The groundbreaking CNN that revolutionized computer vision by winning ImageNet 2012 with a deep architecture and GPU training.

EMNLP 2014

GloVe

Global Vectors for Word Representation - Combining global corpus statistics with local context methods for superior word embeddings.

arXiv 2016

WaveNet

A Generative Model for Raw Audio - DeepMind's autoregressive neural network for generating high-fidelity raw audio waveforms, powering Google Assistant's natural speech synthesis.

ICCV 2021 Best Paper

Swin Transformer

Hierarchical Vision Transformer using Shifted Windows - The Marr Prize winner that brought hierarchical structure and shifted window attention to vision transformers.

NeurIPS 2015

Faster R-CNN

Towards Real-Time Object Detection with Region Proposal Networks - The breakthrough that unified region proposal and detection into a single network, achieving 5 FPS with near cost-free region proposals.

Annals of Statistics 2001

Gradient Boosting

Friedman's seminal framework connecting boosting to gradient descent in function space. The theoretical foundation powering XGBoost, LightGBM, and modern gradient boosting machines.

Machine Learning 2001

Random Forests

Leo Breiman's ensemble masterpiece combining bagging with random feature selection. Creates uncorrelated decision trees that vote together, achieving unprecedented accuracy with built-in OOB validation.

The Annals of Statistics 2001

Gradient Boosting Machine

Jerome Friedman's optimization framework that views boosting as gradient descent in function space. Fits weak learners to pseudo-residuals, enabling any differentiable loss function and spawning XGBoost, LightGBM, and CatBoost.

ICML 1997

AdaBoost

Freund & Schapire's adaptive boosting algorithm that sequentially trains weak learners on reweighted data, focusing on misclassified examples. The first practical boosting method that sparked the ensemble learning revolution.

ICML 2021

EfficientNetV2

Smaller Models and Faster Training - Introduces Fused-MBConv blocks, training-aware NAS, and progressive learning to achieve 11× faster training and state-of-the-art accuracy with fewer parameters.

ICLR 2019

BigGAN

Large Scale GAN Training - Demonstrates that GANs benefit enormously from scaling. 8× batch size and 2-4× parameters yield 3× quality improvement. Introduces truncation trick for fidelity-diversity trade-off.

CVPR 2018 / TPAMI 2020

SENet

Squeeze-and-Excitation Networks - Introduces channel attention mechanism that adaptively recalibrates feature responses. Won ImageNet 2017 with 2.251% top-5 error, a 25% improvement over 2016 winner.

ICLR 2017

NAS

Neural Architecture Search with Reinforcement Learning - The groundbreaking work that launched AutoML. Uses an RNN controller trained with REINFORCE to automatically discover CNN and RNN architectures that rival human design.

ICLR 2015

Bahdanau Attention

Neural Machine Translation by Jointly Learning to Align and Translate. The foundational attention mechanism that gave Seq2Seq models "new life" — introduced dynamic context vectors and soft alignment learning.

NAACL 2018 (Best Paper)

ELMo

Deep Contextualized Word Representations - Bidirectional LSTM-based context-dependent embeddings. First to show that pretrained contextualized representations dramatically improve NLP task performance.

CVPR 2022 (Outstanding Paper)

ConvNeXt

A ConvNet for the 2020s - The pure ConvNet that rivaled Vision Transformers. Starting from ResNet-50, modernizing step-by-step with Transformer-inspired design choices to achieve 87.8% ImageNet accuracy without self-attention.

RL • OpenAI 2017

Proximal Policy Optimization

A simple, scalable, and stable policy gradient method for reinforcement learning - Uses clipped surrogate objective to prevent destructive large policy updates, becoming the go-to algorithm for RL.

NeurIPS 2017

RLHF

Deep Reinforcement Learning from Human Preferences - The foundational methodology that aligns AI with human intent. Powers ChatGPT, InstructGPT, and modern aligned language models through reward model training and PPO optimization.

SOSP 2023

PagedAttention (vLLM)

Efficient Memory Management for Large Language Model Serving - Revolutionary OS-inspired virtual memory system for KV cache. Enables 2-4× throughput vs HuggingFace and near-zero GPU memory waste through dynamic block allocation and memory sharing.

arXiv 2024

Kolmogorov-Arnold Networks

A Revolutionary Alternative to MLPs - Learnable activation functions on edges instead of fixed activations on nodes. Based on the 1957 Kolmogorov-Arnold representation theorem. 10-100× more parameter efficient with faster scaling laws.

ICLR 2022

LoRA

Low-Rank Adaptation of Large Language Models - The parameter-efficient fine-tuning breakthrough that reduces trainable parameters by 10,000× while matching full fine-tuning performance. Zero inference overhead with weight merging.

NeurIPS 2023 (Oral)

DPO

Direct Preference Optimization: Your Language Model is Secretly a Reward Model - A simpler, more efficient alternative to RLHF that eliminates reward modeling and directly optimizes language models from human preferences.

NeurIPS 2019

RMSNorm

Root Mean Square Layer Normalization — A simplified, faster alternative to LayerNorm that removes mean-centering while maintaining training stability. 7-64% speedup with comparable performance. Used in LLaMA, Mistral, and modern LLMs.

NAACL 2018 Best Paper

ELMo

Deep Contextualized Word Representations — The breakthrough that introduced contextualized embeddings, solving the polysemy problem by generating dynamic word vectors from deep bidirectional LSTMs. Each word's representation changes based on its context.

Nature 2014

Neural Turing Machines

Neural network with external memory that can read, write, and access content via attention mechanisms. Extended neural networks with addressable memory, enabling algorithmic learning.

SIGGRAPH 2023

3D Gaussian Splatting

Real-Time Radiance Field Rendering — Revolutionary alternative to NeRF using explicit 3D Gaussian primitives instead of neural networks. Achieves 100-600+ FPS photorealistic rendering with ~1 minute training time vs. days for NeRF.

arXiv 2014

Neural Turing Machine

Neural architecture augmented with external memory - The foundational paper introducing differentiable memory access, enabling neural networks to learn algorithms by reading and writing to an external memory matrix.

ECCV 2018

CBAM

Convolutional Block Attention Module - A lightweight attention module that sequentially applies channel and spatial attention to boost CNN representation power with minimal overhead.

TACL 2017

FastText

Enriching Word Vectors with Subword Information - Efficient text classification and word representation learning using character n-grams, enabling handling of rare and out-of-vocabulary words.

ICLR 2017

GCN

Semi-Supervised Classification with Graph Convolutional Networks - The foundational paper for modern graph neural networks. Introduces a scalable layer-wise propagation rule via spectral graph convolution approximation, enabling efficient semi-supervised learning on graph-structured data.

NeurIPS 2018 (Best Paper)

Neural ODE

Neural Ordinary Differential Equations - Continuous-depth neural networks with constant memory cost. Replaces discrete layers with ODE solvers, enabling adaptive computation and continuous normalizing flows for generative modeling.
