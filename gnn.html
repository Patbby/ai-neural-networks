🕸️ GNN 图神经网络

Graph Neural Networks - Learning from connections and relationships

为什么需要GNN？

传统神经网络（如CNN、RNN）处理的是规则结构数据：图像（网格）、文本（序列）。但现实世界充满了不规则的图结构：社交网络、分子结构、知识图谱、推荐系统...

GNN的核心思想： 通过消息传递(Message Passing)机制，让每个节点聚合邻居信息，逐步更新自身表示。经过多层传播，每个节点都能获得图中更远距离的结构信息。

GNN的通用框架

几乎所有GNN变体都遵循以下两步迭代：

1. 消息生成 (Message)

m_{u→v}^{(l)} = MSG^{(l)}(h_u^{(l-1)}, h_v^{(l-1)}, e_{uv})

邻居节点u向目标节点v发送消息，可基于双方特征和边属性。

2. 聚合与更新 (Aggregate & Update)

h_v^{(l)} = UPDATE^{(l)}(h_v^{(l-1)}, AGG_{u∈N(v)}({m_{u→v}^{(l)}}))

聚合邻居消息，结合自身旧状态，生成新表示。

主要GNN变体

GCN (Graph Convolutional Network)

最经典的GNN，使用对称归一化的邻接矩阵进行特征传播：

H^{(l+1)} = σ(D̃^{-1/2} Ã D̃^{-1/2} H^{(l)} W^{(l)})
import torch.nn.functional as F from torch_geometric.nn import GCNConv class GCN(torch.nn.Module): def __init__(self, in_channels, hidden_channels, out_channels): super().__init__() self.conv1 = GCNConv(in_channels, hidden_channels) self.conv2 = GCNConv(hidden_channels, out_channels) def forward(self, x, edge_index): x = self.conv1(x, edge_index) # 第1层图卷积 x = F.relu(x) x = F.dropout(x, training=self.training) x = self.conv2(x, edge_index) # 第2层图卷积 return F.log_softmax(x, dim=1)

GraphSAGE

采样聚合(SAmple and agGreGatE)，可扩展到大规模图：

h_v^{(l)} = σ(W · CONCAT(h_v^{(l-1)}, AGG({h_u^{(l-1)}, ∀u∈N(v)})))

GIN (Graph Isomorphism Network)

理论上最强大的GNN，能区分更多图结构：

h_v^{(l)} = MLP^{(l)}((1 + ε^{(l)}) · h_v^{(l-1)} + Σ_{u∈N(v)} h_u^{(l-1)})

应用场景

节点级别： 社交网络用户分类、论文引用网络主题预测

图级别： 分子性质预测、程序代码漏洞检测、3D点云分类
