🖼️ CNN 卷积神经网络

Convolutional Neural Networks - How machines learn to "see"

什么是卷积？

卷积神经网络（CNN）是计算机视觉领域的基石。与传统的全连接神经网络不同，CNN利用局部感受野和权值共享来高效处理图像数据。

核心思想： 图像中的局部特征（如边缘、纹理）在图像的任何位置都使用相同的检测器（卷积核）。这种"平移不变性"让CNN能高效学习视觉模式。

核心组件

1. 卷积层 (Convolutional Layer)

卷积层是CNN的核心。它使用一组可学习的滤波器（卷积核）扫描输入数据，提取局部特征。

(I * K)(i, j) = Σ_m Σ_n I(i+m, j+n) · K(m, n)
import torch.nn as nn # 定义卷积层：输入通道=3，输出通道=64，卷积核=3×3 conv = nn.Conv2d(in_channels=3, out_channels=64, kernel_size=3, padding=1) # 前向传播 output = conv(input) # 输入: (B, 3, H, W) → 输出: (B, 64, H, W)

2. 池化层 (Pooling Layer)

池化层降低特征图的空间维度，减少计算量并提供平移不变性。最常用的最大池化(Max Pooling)保留每个区域的最强激活值。

🎯 最大池化演示 (2×2 池化核，步长2)

3. 激活函数 (Activation Function)

引入非线性，让网络能学习复杂模式。ReLU（Rectified Linear Unit）是最常用的激活函数：

ReLU(x) = max(0, x)

经典架构演进

LeNet (1998) → 现代CNN的奠基者，手写数字识别
AlexNet (2012) → ReLU+Dropout，引爆深度学习革命
VGG (2014) → 3×3小卷积核堆叠，证明深度的重要性
ResNet (2015) → 残差连接，可训练152+层网络
EfficientNet (2019) → 复合缩放，效率与精度平衡

总结

CNN通过局部连接、权值共享和空间下采样三大特性，高效处理网格状数据（图像）。理解卷积的本质——局部特征的滑动检测——是掌握计算机视觉的关键。
